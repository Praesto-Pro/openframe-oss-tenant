A React component that renders the main chat interface for the Flamingo AI assistant, providing both real-time messaging and conversation resumption capabilities.

## Key Components

- **ChatView** - Main component that orchestrates the entire chat interface
- **State Management** - Manages current model info, resumable dialogs, and chat metadata
- **Chat Hooks** - Integrates `useChat` for messaging and `useConnectionStatus` for server connectivity
- **Layout Structure** - Uses OpenFrame core components (ChatContainer, ChatHeader, ChatContent, ChatFooter)
- **Resumable Dialogs** - Allows users to continue previous conversations via GraphQL service
- **Quick Actions** - Displays predefined help actions when no messages are present
- **Model Display** - Shows current AI model and provider information

## Usage Example

```typescript
import { ChatView } from './ChatView'

function App() {
  return (
    <div className="app">
      <ChatView />
    </div>
  )
}

// The component automatically handles:
// - Connection status monitoring
// - Message history loading
// - Real-time streaming responses
// - Model switching and display
// - Dialog resumption workflow
```

The component provides a complete chat experience with features like conversation history, quick help actions, model information display, and seamless integration with both API and NATS-based messaging systems. It automatically adapts the UI based on connection status and message state, showing either the welcome screen with quick actions or the full message list interface.
