A GraphQL service that handles dialog and message operations for the Flamingo OpenFrame platform, providing authenticated access to resumable dialogs and their message history.

## Key Components

- **DialogGraphQLService** - Main service class managing GraphQL client lifecycle and authentication
- **ResumableDialog** - Interface for dialog metadata including status, ratings, and timestamps
- **Message/MessageData** - Extended message interfaces with dialog mode support
- **MessagesConnection** - Paginated message collection with cursor-based pagination
- **GraphQL Queries** - Pre-defined queries for fetching dialogs (`GET_RESUMABLE_DIALOG_QUERY`) and messages (`GET_DIALOG_MESSAGES_QUERY`)

## Usage Example

```typescript
import { dialogGraphQLService } from './dialogGraphQLService'

// Get the current resumable dialog
const dialog = await dialogGraphQLService.getResumableDialog()
if (dialog) {
  console.log(`Dialog: ${dialog.title} (${dialog.status})`)
  
  // Fetch all messages for this dialog
  const messages = await dialogGraphQLService.getDialogMessages(dialog.id)
  if (messages) {
    console.log(`Found ${messages.edges.length} messages`)
    
    // Process messages
    messages.edges.forEach(edge => {
      const message = edge.node
      console.log(`${message.owner.type}: ${message.messageData.type}`)
    })
  }
}

// Clean up when done
dialogGraphQLService.dispose()
```

The service automatically handles authentication via `tokenService`, supports cursor-based pagination for large message sets, and provides comprehensive error handling with fallback null returns.
