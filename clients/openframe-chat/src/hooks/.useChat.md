A React hook that provides comprehensive chat functionality with real-time messaging, approvals, and NATS streaming support for the Flamingo OpenFrame chat interface.

## Key Components

- **Core State Management**: Manages typing indicators, streaming status, dialog IDs, and error states
- **Token & API Services**: Integrates with `tokenService` and `ChatApiService` for authentication and API calls
- **Real-time Processing**: Uses NATS WebSocket subscriptions for live message streaming and chunk processing
- **Message Management**: Leverages `useChatMessages` and `useChatApprovals` for message state and approval workflows
- **Chunk Catchup**: Implements `useChunkCatchup` for handling missed messages during reconnection
- **Historical Message Support**: Processes existing conversation history when resuming dialogs

## Usage Example

```typescript
import { useChat } from './useChat'

function ChatComponent() {
  const {
    messages,
    isTyping,
    error,
    sendMessage,
    clearMessages,
    handleQuickAction
  } = useChat({
    useApi: true,
    useNats: true,
    onMetadataUpdate: (metadata) => {
      console.log('Model info:', metadata.modelName, metadata.providerName)
    }
  })

  const handleSend = async () => {
    await sendMessage("Hello, how can you help me?")
  }

  return (
    <div>
      {messages.messages.map(msg => (
        <div key={msg.id}>{msg.content}</div>
      ))}
      {isTyping && <div>Fae is typing...</div>}
      {error && <div className="error">{error}</div>}
      <button onClick={handleSend}>Send Message</button>
    </div>
  )
}
```

The hook handles complex real-time chat scenarios including message streaming, approval requests, error recovery, and dialog resumption while providing a clean interface for chat UI components.
