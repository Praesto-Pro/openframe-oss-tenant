A comprehensive React hook for managing AI chat functionality with real-time messaging, NATS integration, and approval workflows in the Flamingo/OpenFrame platform.

## Key Components

**Main Hook Function:**
- `useChat(options)` - Core chat management hook with configurable API and NATS support

**State Management:**
- Message state via `useChatMessages` and `useChatApprovals` 
- Typing indicators, streaming status, and error handling
- Dialog resumption and incomplete message state tracking

**Real-time Communication:**
- NATS WebSocket subscription for live message streaming
- Chunk processing for streaming responses
- Catchup mechanism for missed messages during reconnection

**Core Functions:**
- `sendMessage(text)` - Send user messages and handle AI responses
- `handleQuickAction(actionText)` - Execute predefined quick actions
- `clearMessages()` - Reset chat state

## Usage Example

```typescript
import { useChat } from './useChat'

function ChatComponent() {
  const { 
    messages, 
    sendMessage, 
    isTyping, 
    error,
    quickActions,
    handleQuickAction 
  } = useChat({
    useNats: true,
    onMetadataUpdate: (metadata) => {
      console.log(`Model: ${metadata.modelName}, Provider: ${metadata.providerName}`)
    }
  })

  const handleSend = async () => {
    await sendMessage("Hello, I need help with my deployment")
  }

  return (
    <div>
      {messages.map(msg => (
        <div key={msg.id}>{msg.content}</div>
      ))}
      {isTyping && <div>Fae is typing...</div>}
      {error && <div className="error">{error}</div>}
      <button onClick={handleSend}>Send Message</button>
    </div>
  )
}
```

The hook integrates with the broader Flamingo ecosystem, supporting approval workflows, token management, and debug modes while providing a clean interface for chat interactions.
