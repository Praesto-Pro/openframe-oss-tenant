Processes historical chat messages from GraphQL queries into standardized chat display format with Fae AI assistant integration. Handles message transformation, error processing, and applies Fae-specific branding and avatars.

## Key Components

- **`processHistoricalMessages`** - Main function that converts GraphQL messages to chat display format
- **Message transformation pipeline** - Maps GraphQL message structure to frontend Message interface
- **Fae AI integration** - Applies Fae assistant branding, avatar, and chat type filtering
- **Error handling support** - Processes messages with error states through core utilities

## Usage Example

```typescript
import { processHistoricalMessages } from './messageProcessor'
import type { Message as GraphQLMessage } from '../services/dialogGraphQLService'

// Process messages from GraphQL query
const graphqlMessages: GraphQLMessage[] = await fetchChatHistory(dialogId)

const displayMessages = processHistoricalMessages(
  graphqlMessages,
  async (requestId) => {
    // Handle approval action
    await approveRequest(requestId)
  },
  async (requestId) => {
    // Handle rejection action  
    await rejectRequest(requestId)
  }
)

// Use processed messages in chat component
<ChatDisplay messages={displayMessages} />
```

The processor ensures consistent message formatting across the application while maintaining Fae AI assistant context and enabling interactive approval/rejection workflows for client-facing chats.
