Module declaration file that organizes and exposes utility modules for system operations including directory management, file permissions, and platform-specific functionality. Provides convenient re-exports of commonly used types and functions.

## Key Components

**Module Exports:**
- `directories` - Directory management utilities
- `file_lock` - File locking detection and management
- `permissions` - Permission checking and capability management
- `uninstall` - Safe directory removal operations
- `update_scripts` - Update script handling
- `updater_launcher` - Application updater functionality
- `user_session` - User session management

**Windows-Specific Modules:**
- `windows_cleanup` - Windows-specific cleanup operations
- `powershell` - PowerShell integration utilities

**Re-exported Items:**
- `DirectoryError`, `DirectoryManager` - Core directory operations
- `Capability`, `PermissionError`, `PermissionUtils`, `Permissions` - Permission system
- `LockingProcess` and file lock utilities (Windows only)
- `remove_directory_with_retry` - Safe uninstall operations
- `get_powershell_path` - PowerShell path resolution (Windows only)

## Usage Example

```rust
use crate::utilities::{
    DirectoryManager, 
    Permissions, 
    Capability,
    remove_directory_with_retry
};

#[cfg(target_os = "windows")]
use crate::utilities::{get_powershell_path, is_file_in_use_error};

// Use directory management
let dir_manager = DirectoryManager::new();

// Check permissions
let perms = Permissions::new();
if perms.has_capability(Capability::WriteFiles) {
    // Perform file operations
}

// Safe directory removal
remove_directory_with_retry(&path)?;
```
