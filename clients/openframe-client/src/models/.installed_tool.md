Represents installed tools and their configuration on managed endpoints within the Flamingo MSP platform. This model tracks tool metadata, execution parameters, and installation status for deployed software across the IT infrastructure.

## Key Components

### ToolStatus Enum
- **Installed**: Indicates the tool is successfully installed and ready for use
- Implements Default trait returning `Installed` status

### InstalledTool Struct
- **tool_agent_id**: Unique identifier for the tool agent instance
- **tool_id**: Base tool identifier
- **tool_type**: Classification of the tool (e.g., monitoring, security, backup)
- **version**: Installed version string
- **session_type**: Execution context (Service, Interactive, etc.)
- **run_command_args**: Command-line arguments for tool execution
- **tool_agent_id_command_args**: Agent-specific command arguments
- **uninstallation_command_args**: Optional removal commands
- **status**: Current installation status
- **executable_path**: Optional relative path to the main executable
- **installation_type**: Method used for installation (MSI, ZIP, etc.)
- **bundle_id**: Optional bundle identifier for grouped installations

## Usage Example

```rust
use crate::models::{InstalledTool, ToolStatus, SessionType};
use crate::models::download_configuration::InstallationType;

// Create a new installed tool record
let tool = InstalledTool {
    tool_agent_id: "monitoring-agent-001".to_string(),
    tool_id: "nagios".to_string(),
    tool_type: "monitoring".to_string(),
    version: "4.4.6".to_string(),
    session_type: SessionType::Service,
    run_command_args: vec!["--config".to_string(), "/etc/nagios.cfg".to_string()],
    tool_agent_id_command_args: vec!["--agent-id".to_string(), "monitoring-agent-001".to_string()],
    status: ToolStatus::Installed,
    executable_path: Some("bin/nagios".to_string()),
    installation_type: InstallationType::Archive,
    ..Default::default()
};

// Serialize for API responses or storage
let json = serde_json::to_string(&tool)?;
```
