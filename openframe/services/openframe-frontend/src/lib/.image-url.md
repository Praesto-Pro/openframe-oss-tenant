A utility function that converts relative image paths to fully qualified URLs using the tenant's API base URL.

## Key Components

- **`getFullImageUrl`** - Main export that transforms image URLs by prepending the tenant host and API path prefix when needed

## Usage Example

```typescript
import { getFullImageUrl } from './image-url'

// Relative path without leading slash
const url1 = getFullImageUrl('uploads/avatar.jpg')
// Returns: "https://tenant.flamingo.run/api/uploads/avatar.jpg"

// Relative path with leading slash
const url2 = getFullImageUrl('/uploads/avatar.jpg') 
// Returns: "https://tenant.flamingo.run/api/uploads/avatar.jpg"

// Already has /api/ prefix
const url3 = getFullImageUrl('/api/uploads/avatar.jpg')
// Returns: "https://tenant.flamingo.run/api/uploads/avatar.jpg"

// Already a full URL
const url4 = getFullImageUrl('https://external.com/image.jpg')
// Returns: "https://external.com/image.jpg" (unchanged)

// Null/undefined input
const url5 = getFullImageUrl(null)
// Returns: undefined
```

The function handles various input formats and intelligently constructs proper API URLs while preserving existing full URLs and handling edge cases like null inputs.
