A specialized API client that extends the base API functionality with comprehensive Tactical RMM integration capabilities. It provides tenant-aware URL construction and a complete set of methods for managing agents, scripts, tasks, and system monitoring.

## Key Components

**Main Class:**
- `TacticalApiClient` - Primary class extending base API client with Tactical RMM endpoints
- `tacticalApiClient` - Singleton instance for application-wide usage

**Core Methods:**
- `buildTacticalUrl()` - Constructs tenant-aware URLs for Tactical endpoints
- HTTP methods (`get`, `post`, `put`, `patch`, `delete`) - Standard REST operations
- Agent management (`getAgents`, `getAgent`, `runScript`, `runBulkAction`)
- Script operations (`getScripts`, `createScript`, `updateScript`)
- System monitoring (`getAgentLogs`, `getAgentSystemInfo`, `getAgentProcesses`)
- Task scheduling (`getScheduledTasks`, `createScheduledTask`, `deleteScheduledTask`)
- Script schedules (`getScriptSchedules`, `createScriptSchedule`, `getScriptScheduleHistory`)

## Usage Example

```typescript
import { tacticalApiClient } from './tactical-api-client'

// Get all agents
const agentsResponse = await tacticalApiClient.getAgents()
if (agentsResponse.success) {
  console.log('Agents:', agentsResponse.data)
}

// Run script on specific agent
const scriptResult = await tacticalApiClient.runScript('agent-123', {
  output: 'json',
  emails: ['admin@company.com'],
  emailMode: 'always',
  custom_field: null,
  save_all_output: true,
  script: 456,
  args: ['--verbose'],
  env_vars: [],
  timeout: 300,
  run_as_user: false,
  run_on_server: false
})

// Create scheduled task
const taskResponse = await tacticalApiClient.createScheduledTask('agent-123', {
  actions: [{
    type: 'script',
    name: 'System Check',
    script: 789,
    timeout: 180,
    script_args: [],
    env_vars: [],
    run_as_user: false
  }],
  name: 'Daily System Check',
  task_type: 'daily',
  run_time_date: '2024-01-15T09:00:00Z',
  daily_interval: 1
})
```
