A specialized API client that extends the base API client with comprehensive Tactical RMM functionality, providing methods for managing agents, scripts, tasks, and system monitoring within the Tactical RMM platform.

## Key Components

- **TacticalApiClient**: Main class that wraps the base API client with Tactical-specific endpoints
- **HTTP Methods**: Standard REST operations (GET, POST, PUT, PATCH, DELETE) with JSON serialization
- **Agent Management**: Methods for retrieving agent data, logs, system info, and running scripts
- **Script Operations**: CRUD operations for managing and executing scripts on agents
- **Task Scheduling**: Methods for creating, managing, and executing scheduled tasks
- **System Monitoring**: Endpoints for accessing agent services, processes, event logs, and system metrics

## Usage Example

```typescript
import { tacticalApiClient } from './tactical-api-client'

// Get all agents
const agentsResponse = await tacticalApiClient.getAgents()
const agents = agentsResponse.data

// Run a script on a specific agent
const scriptResult = await tacticalApiClient.runScript('agent-123', {
  output: 'wait',
  emails: ['admin@company.com'],
  emailMode: 'default',
  custom_field: null,
  save_all_output: true,
  script: 456,
  args: ['--verbose'],
  env_vars: [],
  timeout: 300,
  run_as_user: false,
  run_on_server: false
})

// Create a scheduled daily task
const task = await tacticalApiClient.createScheduledTask('agent-123', {
  name: 'Daily System Check',
  task_type: 'daily',
  run_time_date: '09:00:00',
  daily_interval: 1,
  actions: [{
    type: 'script',
    name: 'System Health Check',
    script: 789,
    timeout: 600,
    script_args: [],
    env_vars: [],
    run_as_user: false
  }]
})
```
