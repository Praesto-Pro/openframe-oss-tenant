A comprehensive React table component for displaying and managing logs with filtering, pagination, and detailed view capabilities.

## Key Components

- **LogsTable**: Main forwardRef component that displays logs in a paginated table format
- **UILogEntry**: TypeScript interface defining the structure for transformed log entries
- **LogsTableRef**: Ref interface exposing refresh functionality
- **Column definitions**: Configurable table columns for log ID, status, tool, source, and description
- **Filter integration**: Real-time filtering by severity, tool type, and organization
- **Modal integration**: Side panel for detailed log information via LogInfoModal

## Usage Example

```typescript
import { LogsTable, type LogsTableRef } from './logs-table'

// Basic usage
function LogsPage() {
  return <LogsTable />
}

// Embedded mode for device-specific logs
function DeviceLogsSection() {
  return <LogsTable deviceId="device-123" embedded={true} />
}

// With ref for external refresh control
function LogsWithRefresh() {
  const logsRef = useRef<LogsTableRef>(null)
  
  const handleRefresh = () => {
    logsRef.current?.refresh()
  }

  return (
    <div>
      <button onClick={handleRefresh}>Refresh</button>
      <LogsTable ref={logsRef} />
    </div>
  )
}
```

The component supports both standalone and embedded modes, with automatic URL parameter synchronization for filters and pagination. It transforms raw API log entries into a user-friendly format with status badges, tool indicators, and device information.
