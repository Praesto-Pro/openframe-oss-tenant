This React component provides a remote desktop interface that connects to devices via MeshCentral, enabling real-time desktop control and management within the Flamingo Stack platform.

## Key Components

- **RemoteDesktopPage**: Main page component that handles device connection and remote desktop functionality
- **MeshCentral Integration**: Uses `MeshControlClient`, `MeshTunnel`, and `MeshDesktop` for establishing connections
- **Device Data Handling**: Supports both legacy query parameters and internal device fetching for backward compatibility
- **Connection Management**: Handles WebSocket tunneling, reconnection logic, and connection state tracking
- **Input Control**: Manages keyboard shortcuts, power actions, and display switching
- **Settings Modal**: Provides configuration options through `RemoteSettingsModal`

## Usage Example

This component is typically accessed through device routing in the Flamingo Stack application:

```typescript
// Route: /devices/[deviceId]/remote-desktop
// The component automatically handles:
// - Device authentication and connection
// - Canvas-based desktop rendering
// - Real-time input forwarding
// - Connection state management

// Legacy usage with query parameters (backward compatibility):
const legacyUrl = `/devices/${deviceId}/remote-desktop?deviceData=${JSON.stringify({
  id: 'device-123',
  meshcentralAgentId: 'agent-456',
  hostname: 'workstation-01',
  organization: 'Acme Corp'
})}`

// Modern usage relies on internal device fetching:
const modernUrl = `/devices/${deviceId}/remote-desktop`
```

The component provides comprehensive remote desktop functionality including keyboard shortcuts (Ctrl+Alt+Del), power management, display switching, and real-time connection monitoring with automatic reconnection capabilities.
