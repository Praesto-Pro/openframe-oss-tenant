Modal component for configuring remote desktop connection settings including quality, scaling, frame rate, and input preferences. Integrates with MeshCentral desktop and tunnel components to apply settings in real-time.

## Key Components

- **RemoteSettingsModal** - Main modal component for remote desktop settings configuration
- **RemoteSettingsModalProps** - TypeScript interface defining modal props including settings, refs, and callbacks
- **Settings Management** - Handles validation, application, and persistence of remote desktop preferences
- **UI Controls** - Select dropdowns for quality/scaling/framerate and checkboxes for input options

## Usage Example

```typescript
import { RemoteSettingsModal } from './remote-settings-modal'
import { RemoteSettingsConfig } from '@lib/meshcentral/remote-settings'

function RemoteDesktopPage() {
  const [isSettingsOpen, setIsSettingsOpen] = useState(false)
  const [currentSettings, setCurrentSettings] = useState<RemoteSettingsConfig>({
    quality: 80,
    scaling: 100,
    frameRate: 'auto',
    swapMouseButtons: false,
    useRemoteKeyboardMap: false,
    invertScrollDirection: false
  })
  const desktopRef = useRef<MeshDesktop | null>(null)
  const tunnelRef = useRef<MeshTunnel | null>(null)
  const [connectionState, setConnectionState] = useState(0)

  return (
    <>
      <button onClick={() => setIsSettingsOpen(true)}>
        Open Settings
      </button>
      
      <RemoteSettingsModal
        open={isSettingsOpen}
        onOpenChange={setIsSettingsOpen}
        currentSettings={currentSettings}
        desktopRef={desktopRef}
        tunnelRef={tunnelRef}
        connectionState={connectionState}
        onSettingsChange={setCurrentSettings}
      />
    </>
  )
}
```

The modal validates connection state before applying settings and provides toast feedback for success/error states. Settings are immediately applied to the active MeshDesktop instance and persisted via the RemoteDesktopSettings manager.
