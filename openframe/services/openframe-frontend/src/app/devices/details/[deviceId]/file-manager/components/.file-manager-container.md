A React container component that provides a complete file management interface for remote devices using MeshCentral connections. It handles file operations, search, uploads, downloads, and modal interactions with progress tracking and keyboard shortcuts.

## Key Components

- **FileManagerContainer** - Main container component that orchestrates file management operations
- **useMeshFileManager** - Custom hook for MeshCentral file system operations
- **Modal Components** - NewFolderModal, RenameItemModal, DeleteConfirmationModal for user interactions
- **Progress Tracking** - Real-time upload/download progress indicators
- **Keyboard Shortcuts** - Ctrl+C/X/V for copy/cut/paste, Delete key for deletion, Escape for deselection

## Usage Example

```typescript
import { FileManagerContainer } from './file-manager-container'

// Basic usage for device file management
<FileManagerContainer
  deviceId="device-123"
  meshcentralAgentId="agent-456"
  hostname="workstation-01"
  organizationName="Acme Corp"
/>

// Minimal setup
<FileManagerContainer
  deviceId="device-123"
  meshcentralAgentId="agent-456"
/>
```

## Key Features

- **File Operations**: Create folders, rename, delete, upload, download files
- **Navigation**: Breadcrumb navigation, search functionality, path traversal
- **Selection**: Multi-file selection with checkboxes and keyboard shortcuts
- **Clipboard**: Copy/cut/paste operations with visual feedback
- **Progress Tracking**: Real-time upload/download progress with cancellation
- **Modal Management**: Contextual modals for folder creation, renaming, and deletion confirmation
- **Responsive UI**: Skeleton loading states and adaptive layouts
- **Error Handling**: Connection state management and graceful degradation

The component integrates with the Flamingo Stack's OpenFrame file management system to provide seamless remote device file access through MeshCentral agents.
