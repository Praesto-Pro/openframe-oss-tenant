A React hook that fetches and manages the current release version from the API using React Query, with automatic error handling and toast notifications.

## Key Components

- **`useReleaseVersion`** - Main hook for fetching release version data
- **`releaseVersionQueryKeys`** - Structured query key factory for React Query
- **`fetchReleaseVersion`** - Async function that retrieves version from `/api/release-version`
- **`ReleaseVersionResponse`** - Type definition for API response structure
- **`UseReleaseVersionOptions`** - Configuration interface with enabled flag

## Usage Example

```typescript
import { useReleaseVersion } from './use-release-version'

function VersionDisplay() {
  const { releaseVersion, isLoading, error, refetch } = useReleaseVersion({
    enabled: true
  })

  if (isLoading) return <div>Loading version...</div>
  if (error) return <div>Error: {error}</div>

  return (
    <div>
      <p>Current Release: {releaseVersion}</p>
      <button onClick={() => refetch()}>Refresh Version</button>
    </div>
  )
}

// Disabled query example
function ConditionalVersion({ shouldFetch }: { shouldFetch: boolean }) {
  const { releaseVersion } = useReleaseVersion({ 
    enabled: shouldFetch 
  })
  
  return <span>Version: {releaseVersion}</span>
}
```

The hook includes built-in caching (5min stale time, 10min garbage collection), automatic retry logic, and error toast notifications when enabled.
