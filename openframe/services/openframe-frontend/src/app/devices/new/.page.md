A Next.js page component that provides a device installation interface for the OpenFrame platform, allowing users to generate and execute platform-specific installation commands for new devices.

## Key Components

**Main Component:**
- `NewDevicePage` - React component that renders the device setup interface

**State Management:**
- Organization selection with auto-selection of default organization
- OS platform selection (Windows, macOS, Linux)
- Custom argument management for installation commands
- Registration secret and release version handling

**Key Features:**
- Platform-specific binary URL generation
- Dynamic command construction based on selected options
- Script file generation and download for direct execution
- Clipboard operations for commands and paths
- Antivirus exclusion path display
