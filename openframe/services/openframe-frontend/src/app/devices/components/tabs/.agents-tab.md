A React component that displays management agents installed on a device, combining data from installed agents and tool connections to show a unified view of agent status and connectivity.

## Key Components

- **AgentsTab**: Main component that renders agent information in a responsive grid layout
- **agentTypeToToolType**: Mapping object that converts agent types to standardized tool types
- **combinedAgents**: Merged data structure combining installed agents with their corresponding tool connections
- **connectionMap**: Map for efficient lookup of tool connections by tool type

## Usage Example

```typescript
import { AgentsTab } from './agents-tab'
import type { Device } from '../../types/device.types'

function DeviceDetailPage({ deviceData }: { deviceData: Device }) {
  return (
    <div className="device-details">
      <AgentsTab device={deviceData} />
    </div>
  )
}

// Example device data structure
const exampleDevice: Device = {
  installedAgents: [
    {
      agentType: 'tacticalrmm-agent',
      version: '2.5.0'
    }
  ],
  toolConnections: [
    {
      toolType: 'TACTICAL_RMM',
      agentToolId: 'agent-12345'
    }
  ]
}
```

The component handles missing or malformed data gracefully, displays agents with tool connections first, and provides contextual tooltips explaining each agent's purpose and status. When no agents are found, it shows an informative empty state.
