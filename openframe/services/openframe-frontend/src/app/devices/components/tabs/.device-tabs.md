A client-side React configuration module that defines the complete tab structure for device detail views in the Flamingo/OpenFrame platform, including hardware monitoring, security, compliance, and management interfaces.

## Key Components

- **`DEVICE_TABS`** - Main configuration array containing 9 tab definitions with icons, labels, and components
- **`getDeviceTab()`** - Utility function to retrieve a specific tab configuration by ID
- **`getTabComponent()`** - Helper function to extract the React component for a given tab ID

## Usage Example

```typescript
import { DEVICE_TABS, getDeviceTab, getTabComponent } from './device-tabs'

// Render all available tabs
const DeviceTabsNavigation = () => {
  return (
    <nav>
      {DEVICE_TABS.map(tab => (
        <button key={tab.id}>
          <tab.icon />
          {tab.label}
        </button>
      ))}
    </nav>
  )
}

// Get specific tab configuration
const securityTab = getDeviceTab('security')
const SecurityComponent = getTabComponent('security')

// Dynamic tab rendering
const renderTabContent = (activeTabId: string, device: any) => {
  const TabComponent = getTabComponent(activeTabId)
  return TabComponent ? <TabComponent device={device} /> : null
}
```

The module centralizes tab configuration for consistent device management UI across hardware specs, network settings, security posture, compliance status, installed agents, user access, software inventory, vulnerability assessments, and system logs.
