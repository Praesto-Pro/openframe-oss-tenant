A React modal component for selecting and executing tactical scripts on managed devices with platform-specific filtering and batch execution capabilities.

## Key Components

- **ScriptsModal**: Main modal component with script selection and execution interface
- **CustomCheckbox**: Custom checkbox component for script selection UI
- **filterScriptsByPlatform**: Utility function that filters scripts based on device OS compatibility
- **getCategoriesFromScripts**: Helper function to extract and sort script categories

## Usage Example

```typescript
import { ScriptsModal } from './scripts-modal'

function DeviceActions({ device }: { device: Device }) {
  const [isScriptsModalOpen, setIsScriptsModalOpen] = useState(false)
  
  const handleRunScripts = (scriptIds: string[]) => {
    console.log('Running scripts:', scriptIds)
  }
  
  const handleDeviceLogs = () => {
    // Navigate to device logs
  }
  
  return (
    <ScriptsModal
      isOpen={isScriptsModalOpen}
      onClose={() => setIsScriptsModalOpen(false)}
      deviceId={device.id}
      device={device}
      onRunScripts={handleRunScripts}
      onDeviceLogs={handleDeviceLogs}
    />
  )
}
```

The modal provides search functionality, category filtering, platform compatibility checking, and batch script execution with toast notifications and automatic redirection to device logs upon completion.
