A React component that renders a detailed view page for individual devices with comprehensive device information, actions, and tabbed navigation.

## Key Components

- **DeviceDetailsView**: Main component that displays device details with header actions and tabbed content
- **DeviceInfoSection**: Shows basic device information in a structured layout
- **DeviceActionsDropdown**: Provides additional device management actions
- **ScriptsModal**: Modal for running scripts on the device
- **TabNavigation/TabContent**: Tabbed interface for organizing device information
- **Header Actions**: Buttons for remote control, shell access, and file management

## Usage Example

```typescript
import { DeviceDetailsView } from './device-details-view'

function DevicePage({ params }: { params: { id: string } }) {
  return (
    <DeviceDetailsView deviceId={params.id} />
  )
}

// The component automatically handles:
// - Device data fetching via useDeviceDetails hook
// - URL parameter processing (e.g., ?action=runScript)
// - Platform-specific shell options (CMD/PowerShell for Windows)
// - Action availability based on device capabilities
// - Tab navigation with URL synchronization
```

The component features automatic refresh for relative timestamps, handles URL-based actions, provides platform-specific remote shell options, and includes comprehensive error handling with loading states.
