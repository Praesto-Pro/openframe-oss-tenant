Generates platform-specific installation and uninstallation commands for OpenFrame client devices across macOS, Windows, and Linux platforms.

## Key Components

- **URL Builders**: `buildBinaryUrl()`, `getMacBinaryUrl()`, `getWindowsBinaryUrl()` - Generate GitHub release download URLs
- **Command Builders**: `buildInstallCommand()`, `buildUninstallCommand()` - Create platform-specific shell commands for device management
- **Platform Utilities**: `normalizeDevicePlatform()` - Standardize OS detection from various device fields
- **Type Definitions**: `InstallCommandOptions`, `UninstallCommandOptions` - Configuration interfaces for command generation

## Usage Example

```typescript
import { buildInstallCommand, normalizeDevicePlatform } from './device-command-utils'

// Generate installation command for a Windows device
const installCmd = buildInstallCommand({
  platform: 'windows',
  serverUrl: 'https://api.flamingo.run',
  initialKey: 'ak_1234567890abcdef',
  orgId: 'org_xyz123',
  releaseVersion: 'v1.2.3',
  additionalArgs: ['--debug', '--force']
})

// Normalize device platform from various OS strings
const platform = normalizeDevicePlatform('Win64') // Returns 'windows'
const macPlatform = normalizeDevicePlatform('macOS Big Sur') // Returns 'darwin'

// Generate uninstall command
const uninstallCmd = buildUninstallCommand({
  platform: 'darwin',
  releaseVersion: 'latest'
})
```

The utility handles cross-platform differences in command syntax, binary extraction, and privilege escalation (PowerShell for Windows, bash with sudo for macOS/Linux).
