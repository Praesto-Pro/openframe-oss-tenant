Utility functions for building OpenFrame client device installation and uninstallation commands across different operating systems (macOS, Windows).

## Key Components

- **Binary URL Functions**: `buildBinaryUrl()`, `getMacBinaryUrl()`, `getWindowsBinaryUrl()` - Generate download URLs for OpenFrame client binaries
- **Command Builders**: `buildInstallCommand()`, `buildUninstallCommand()` - Create platform-specific shell commands for device management
- **Platform Normalization**: `normalizeDevicePlatform()` - Standardize OS platform identifiers from various device fields
- **Type Interfaces**: `InstallCommandOptions`, `UninstallCommandOptions` - Type-safe configuration for command generation

## Usage Example

```typescript
import { buildInstallCommand, buildUninstallCommand, normalizeDevicePlatform } from './device-command-utils'

// Generate installation command for Windows
const installCmd = buildInstallCommand({
  platform: 'windows',
  serverUrl: 'https://api.openframe.ai',
  initialKey: 'ak_123456',
  orgId: 'org_789',
  releaseVersion: 'v1.0.0',
  additionalArgs: ['--verbose']
})

// Generate uninstall command for macOS
const uninstallCmd = buildUninstallCommand({
  platform: 'darwin',
  releaseVersion: 'v1.0.0'
})

// Normalize platform from device data
const platform = normalizeDevicePlatform('Windows 11', 'win64', 'Microsoft Windows')
// Returns: 'windows'
```

The utility handles platform-specific binary downloads, PowerShell commands for Windows, and bash commands for macOS/Linux systems.
