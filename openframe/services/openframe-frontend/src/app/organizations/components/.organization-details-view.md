A React component that displays comprehensive organization details in a structured layout with edit and delete functionality.

## Key Components

- **OrganizationDetailsView** - Main component that renders organization details with header actions
- **useOrganizationDetails** - Custom hook for fetching organization data
- **useDeleteOrganization** - Custom hook for organization deletion
- **DetailPageContainer** - Layout wrapper with navigation and header actions
- **InfoCard** - Reusable component for displaying contact and service information
- **OrganizationIcon** - Component for displaying organization images (feature-flagged)

## Usage Example

```typescript
import { OrganizationDetailsView } from './organization-details-view'

// Basic usage in a page component
export default function OrganizationPage({ params }: { params: { id: string } }) {
  return <OrganizationDetailsView id={params.id} />
}

// The component handles:
// - Loading states with CardLoader
// - Error states with LoadError/NotFoundError
// - Organization data display in structured sections
// - Edit/delete actions with toast notifications
// - Feature-flagged image display
// - Responsive grid layouts for contacts and details
```

The component provides a complete organization management interface with summary information, contact details (primary, billing, technical), service configuration, and notes section. It includes proper error handling, loading states, and prevents deletion of default organizations.
