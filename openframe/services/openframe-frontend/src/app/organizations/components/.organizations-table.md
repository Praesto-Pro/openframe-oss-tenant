A React component that renders a paginated, searchable table of organizations with filtering capabilities and navigation to detail/edit pages.

## Key Components

- **OrganizationsTable**: Main component implementing the organizations list view
- **OrganizationNameCell**: Custom cell renderer displaying organization name with optional logo
- **UIOrganizationEntry**: TypeScript interface defining the table row data structure

The component integrates cursor-based pagination, real-time search, client-side filtering by tier/industry, and URL state management for maintaining filter/search state across navigation.

## Usage Example

```typescript
import { OrganizationsTable } from './organizations-table'

// Basic usage in a page component
export default function OrganizationsPage() {
  return (
    <div>
      <OrganizationsTable />
    </div>
  )
}

// The component automatically handles:
// - URL params: ?search=acme&tier[]=enterprise&industry[]=tech
// - Cursor pagination: ?cursor=eyJpZCI6IjEyMyJ9
// - Navigation to /organizations/details/:id on row click
// - Navigation to /organizations/edit/new on add button click
```

The table displays organization name (with logo if enabled), tier/industry, MRR, and last activity. It supports responsive design with columns that hide on smaller screens and includes skeleton loading states during data fetching.
