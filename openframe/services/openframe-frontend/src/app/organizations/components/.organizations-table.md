A React table component that displays paginated organization data with filtering, search, and navigation capabilities for the OpenFrame platform.

## Key Components

- **OrganizationsTable** - Main table component with search, filtering, and pagination
- **OrganizationNameCell** - Custom cell renderer for organization names with optional images
- **UIOrganizationEntry** - TypeScript interface defining the transformed organization data structure

## Usage Example

```typescript
import { OrganizationsTable } from './organizations-table'

// Use in a page component
export default function OrganizationsPage() {
  return <OrganizationsTable />
}
```

The component integrates with:
- **URL-based filtering** for tiers and industries via `useApiParams`
- **Cursor pagination** with `useCursorPaginationState` and server-side data fetching
- **Custom hooks** like `useOrganizations` for data management
- **Feature flags** for conditional image display
- **Navigation** to organization details and edit pages

Key features include real-time search, MRR display formatting, last activity timestamps, responsive column hiding, and integrated "Add Organization" functionality. The table automatically handles loading states, error display, and empty state messaging.
