TypeScript type definitions and utilities for managing script schedules, including task scheduling, execution history, and form validation.

## Key Components

**API Response Types:**
- `ScriptScheduleListItem` - Schedule overview data
- `ScriptScheduleDetail` - Complete schedule configuration
- `ScriptScheduleAction` - Individual script action definition
- `ScriptScheduleAgent` - Agent information for schedule execution
- `ScriptScheduleHistoryEntry` - Execution history records

**Form Types:**
- `CreateScheduleFormData` - Zod-validated form structure
- `ScheduleActionFormData` - Script action form fields
- `ScriptScheduleTaskType` - Schedule frequency types (`runonce`, `daily`, `weekly`, `monthly`)

**Constants:**
- `WEEKDAYS` - Weekday definitions with bit flags
- `PLATFORMS` - Supported operating systems
- `REPEAT_PERIOD_OPTIONS` - UI options for schedule intervals

**Utility Functions:**
- `toZuluDateString()` / `parseZuluToLocalDate()` - UTC date handling
- `scheduleDetailToFormData()` - Convert API data to form format
- `buildCreatePayload()` - Transform form data to API payload
- `getRepeatLabel()` - Human-readable schedule frequency

## Usage Example

```typescript
import { CreateScheduleFormData, buildCreatePayload, toZuluDateString } from './script-schedule.types'

// Create a new schedule
const formData: CreateScheduleFormData = {
  name: 'Daily System Check',
  scheduledDate: new Date(),
  repeatEnabled: true,
  repeatInterval: 1,
  repeatPeriod: 'day',
  weekdays: 0,
  supportedPlatforms: ['windows', 'linux'],
  enabled: true,
  actions: [{
    script: 123,
    name: 'System Health',
    timeout: 300,
    script_args: [],
    env_vars: []
  }]
}

// Convert to API payload
const payload = buildCreatePayload(formData)

// Format date for API
const zuluTime = toZuluDateString(new Date())
```

The file provides comprehensive type safety for schedule management, form validation with Zod schemas, and utility functions for date handling and data transformation between UI and API formats.
