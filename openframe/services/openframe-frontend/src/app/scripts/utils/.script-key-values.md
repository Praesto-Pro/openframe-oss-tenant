Utility functions for parsing and serializing script arguments between key-value string arrays and structured ScriptArgument objects used in OpenFrame's script execution system.

## Key Components

- `parseKeyValues()` - Converts string array of key=value pairs into ScriptArgument objects with auto-generated IDs
- `serializeKeyValues()` - Converts ScriptArgument objects back to key=value string array format

## Usage Example

```typescript
import { parseKeyValues, serializeKeyValues } from './script-key-values'

// Parse command line style arguments
const args = parseKeyValues(['name=john', 'age=30', 'verbose'])
// Returns: [
//   { id: 'arg-0', key: 'name', value: 'john' },
//   { id: 'arg-1', key: 'age', value: '30' },
//   { id: 'arg-2', key: 'verbose', value: '' }
// ]

// Serialize back to string format
const serialized = serializeKeyValues(args)
// Returns: ['name=john', 'age=30', 'verbose']

// Filter out empty keys during serialization
const withEmpty = [{ id: 'arg-0', key: '', value: 'ignored' }]
serializeKeyValues(withEmpty) // Returns: []
```

The functions handle edge cases like missing values (treated as flags), empty arrays, and filter out entries with empty keys during serialization.
