Custom React hooks for managing script schedule mutations using TanStack Query. Provides CRUD operations for script schedules and agent assignment management with automatic cache invalidation.

## Key Components

**API Functions:**
- `createScriptScheduleApi` - Creates new script schedules
- `updateScriptScheduleApi` - Updates existing schedule details
- `deleteScriptScheduleApi` - Removes script schedules
- `replaceScheduleAgentsApi` - Replaces all agents for a schedule
- `addScheduleAgentsApi` - Adds agents to existing schedule
- `removeScheduleAgentsApi` - Removes specific agents from schedule

**React Hooks:**
- `useCreateScriptSchedule` - Mutation hook for creating schedules
- `useUpdateScriptSchedule` - Mutation hook for updating schedules
- `useDeleteScriptSchedule` - Mutation hook for deleting schedules
- `useReplaceScheduleAgents` - Mutation hook for replacing agent assignments
- `useAddScheduleAgents` - Mutation hook for adding agents
- `useRemoveScheduleAgents` - Mutation hook for removing agents

## Usage Example

```typescript
import { 
  useCreateScriptSchedule, 
  useUpdateScriptSchedule, 
  useReplaceScheduleAgents 
} from './use-script-schedule-mutations'

function ScheduleManager() {
  const createMutation = useCreateScriptSchedule()
  const updateMutation = useUpdateScriptSchedule()
  const replaceAgentsMutation = useReplaceScheduleAgents()

  const handleCreateSchedule = async () => {
    try {
      await createMutation.mutateAsync({
        name: 'Daily Backup',
        script_id: 'script-123',
        cron_expression: '0 2 * * *'
      })
    } catch (error) {
      console.error('Failed to create schedule:', error)
    }
  }

  const handleUpdateAgents = async (scheduleId: string) => {
    await replaceAgentsMutation.mutateAsync({
      id: scheduleId,
      agents: ['agent-1', 'agent-2']
    })
  }

  return (
    <div>
      <button 
        onClick={handleCreateSchedule}
        disabled={createMutation.isPending}
      >
        Create Schedule
      </button>
    </div>
  )
}
```
