A custom React hook for managing scripts data using TanStack Query, providing scripts fetching, caching, and state management functionality.

## Key Components

- **scriptsQueryKeys**: Query key factory for TanStack Query cache management
- **fetchAllScripts()**: Async function that retrieves scripts from the tactical API client
- **useScripts()**: Main hook that provides scripts data, loading states, and cache control methods

## Usage Example

```typescript
import { useScripts } from './hooks/use-scripts'

function ScriptsComponent() {
  const { scripts, isLoading, error, refetch, invalidate } = useScripts()

  if (isLoading) return <div>Loading scripts...</div>
  if (error) return <div>Error: {error}</div>

  return (
    <div>
      <button onClick={() => refetch()}>Refresh</button>
      <button onClick={() => invalidate()}>Clear Cache</button>
      
      {scripts.map((script) => (
        <div key={script.id}>
          <h3>{script.name}</h3>
          <p>{script.description}</p>
        </div>
      ))}
    </div>
  )
}
```

The hook returns an array of `ScriptEntry` objects along with standard query states (loading, error) and methods to refresh data or invalidate the cache. It handles error cases by throwing descriptive error messages when the API request fails.
