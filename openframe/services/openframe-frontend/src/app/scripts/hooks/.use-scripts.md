A React Query hook for managing tactical API scripts data with caching and state management.

## Key Components

- **scriptsQueryKeys** - Query key factory for React Query caching
- **fetchAllScripts()** - API function that retrieves script entries from tactical API client
- **useScripts()** - Custom hook providing scripts data with loading states and cache management

## Usage Example

```typescript
import { useScripts } from './hooks/use-scripts'

function ScriptsComponent() {
  const { scripts, isLoading, error, refetch, invalidate } = useScripts()

  if (isLoading) return <div>Loading scripts...</div>
  if (error) return <div>Error: {error}</div>

  return (
    <div>
      <button onClick={() => refetch()}>Refresh Scripts</button>
      <button onClick={() => invalidate()}>Clear Cache</button>
      
      {scripts.map(script => (
        <div key={script.id}>{script.name}</div>
      ))}
    </div>
  )
}
```

The hook automatically handles error states, loading indicators, and provides both refetch (re-run query) and invalidate (clear cache) methods for manual data management. All API responses are typed using the `ScriptEntry` interface from the scripts store.
