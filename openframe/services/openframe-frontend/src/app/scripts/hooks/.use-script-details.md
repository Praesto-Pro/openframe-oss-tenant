React Query hook for fetching detailed script information from the Tactical RMM API, providing script metadata, execution parameters, and platform compatibility details.

## Key Components

- **`ScriptDetails`** - TypeScript interface defining the complete script data structure including execution parameters, platform support, and metadata
- **`scriptDetailsQueryKeys`** - Query key factory for consistent caching and invalidation patterns
- **`fetchScriptDetails()`** - API function that retrieves script details using the tactical API client
- **`useScriptDetails()`** - Main React hook that manages script details fetching with loading states and error handling

## Usage Example

```typescript
import { useScriptDetails } from './hooks/use-script-details'

function ScriptDetailsPanel({ scriptId }: { scriptId: string }) {
  const { scriptDetails, isLoading, error, refetch } = useScriptDetails(scriptId)

  if (isLoading) return <div>Loading script details...</div>
  if (error) return <div>Error: {error}</div>
  if (!scriptDetails) return <div>Script not found</div>

  return (
    <div>
      <h2>{scriptDetails.name}</h2>
      <p>{scriptDetails.description}</p>
      <div>
        <strong>Shell:</strong> {scriptDetails.shell}
        <strong>Category:</strong> {scriptDetails.category}
        <strong>Timeout:</strong> {scriptDetails.default_timeout}s
        <strong>Platforms:</strong> {scriptDetails.supported_platforms.join(', ')}
      </div>
      <pre>{scriptDetails.script_body}</pre>
      <button onClick={() => refetch()}>Refresh</button>
    </div>
  )
}
```

The hook automatically handles caching, loading states, and provides easy access to script execution details including supported platforms, environment variables, and script content.
