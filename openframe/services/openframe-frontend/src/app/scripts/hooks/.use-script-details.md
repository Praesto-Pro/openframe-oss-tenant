A React Query hook for fetching detailed information about tactical RMM scripts, providing type-safe access to script metadata, configuration, and execution details.

## Key Components

- **`ScriptDetails`** - TypeScript interface defining the complete structure of a script object including metadata, execution settings, and platform support
- **`scriptDetailsQueryKeys`** - Query key factory for React Query cache management with hierarchical key structure
- **`fetchScriptDetails()`** - API function that retrieves script details from the tactical API client with error handling
- **`useScriptDetails()`** - Custom React hook that wraps React Query functionality for fetching and caching script details

## Usage Example

```typescript
import { useScriptDetails } from './use-script-details'

function ScriptDetailsComponent({ scriptId }: { scriptId: string }) {
  const { scriptDetails, isLoading, error, refetch } = useScriptDetails(scriptId)

  if (isLoading) return <div>Loading script details...</div>
  if (error) return <div>Error: {error}</div>
  if (!scriptDetails) return <div>Script not found</div>

  return (
    <div>
      <h2>{scriptDetails.name}</h2>
      <p>{scriptDetails.description}</p>
      <div>
        <strong>Shell:</strong> {scriptDetails.shell}
        <strong>Category:</strong> {scriptDetails.category}
        <strong>Timeout:</strong> {scriptDetails.default_timeout}s
      </div>
      <button onClick={() => refetch()}>Refresh</button>
    </div>
  )
}
```

The hook automatically handles caching, loading states, and provides a clean API for accessing script configuration data throughout the application.
