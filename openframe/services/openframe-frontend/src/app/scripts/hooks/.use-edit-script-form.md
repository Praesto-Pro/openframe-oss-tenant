A React hook that manages form state and API operations for creating and editing scripts in the tactical management system.

## Key Components

- **`useEditScriptForm`** - Main hook that returns form instance, submission state, and save handler
- **`createScriptApi`** - API function for creating new scripts
- **`updateScriptApi`** - API function for updating existing scripts  
- **`ScriptPayload`** - TypeScript interface defining the script data structure
- **Form validation** - Uses Zod schema resolver for client-side validation
- **Mutation handling** - Manages create/update operations with React Query

## Usage Example

```typescript
import { useEditScriptForm } from './use-edit-script-form'

function EditScriptDialog({ scriptId, scriptDetails, isEditMode }) {
  const { form, isSubmitting, handleSave } = useEditScriptForm({
    scriptId,
    scriptDetails, 
    isEditMode
  })

  return (
    <form>
      <input {...form.register('name')} />
      <textarea {...form.register('script_body')} />
      <button 
        onClick={handleSave}
        disabled={isSubmitting}
      >
        {isEditMode ? 'Update' : 'Create'} Script
      </button>
    </form>
  )
}
```

The hook automatically populates form fields when editing existing scripts, handles argument and environment variable parsing, manages loading states, and provides success/error toast notifications. It invalidates relevant query cache entries after successful operations to keep the UI synchronized.
