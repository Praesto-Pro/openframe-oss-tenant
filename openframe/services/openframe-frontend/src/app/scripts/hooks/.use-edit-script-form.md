React hook for managing script creation and editing forms with validation, API integration, and toast notifications. Handles form state, data transformation, and submission for both create and update operations.

## Key Components

- **useEditScriptForm**: Main hook that returns form instance, submission state, and save handler
- **Form Management**: Uses react-hook-form with Zod validation for type-safe form handling
- **Data Transformation**: Converts between API format and form-friendly key-value pairs for args and env vars
- **API Integration**: Handles both script creation and updates via tacticalApiClient
- **Error Handling**: Provides comprehensive validation and submission error feedback via toast notifications

## Usage Example

```typescript
import { useEditScriptForm } from './use-edit-script-form'
import { useScriptDetails } from './use-script-details'

function EditScriptDialog({ scriptId, isOpen }: EditScriptDialogProps) {
  const { scriptDetails, isLoading } = useScriptDetails(scriptId)
  const { form, isSubmitting, handleSave } = useEditScriptForm({
    scriptId,
    scriptDetails,
    isEditMode: Boolean(scriptId)
  })

  return (
    <form onSubmit={(e) => { e.preventDefault(); handleSave(); }}>
      {/* Form fields using form.register, form.control, etc. */}
      <Button type="submit" disabled={isSubmitting}>
        {scriptId ? 'Update Script' : 'Create Script'}
      </Button>
    </form>
  )
}
```

The hook automatically populates form fields when editing existing scripts and transforms data between the form's key-value structure and the API's string array format for arguments and environment variables.
