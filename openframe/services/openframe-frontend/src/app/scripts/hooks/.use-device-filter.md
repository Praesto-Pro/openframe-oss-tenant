A React hook that provides device filtering capabilities with search and organization-based filtering for MSP device management.

## Key Components

- **useDeviceFilter** - Main hook that combines search and organization filtering
- **UseDeviceFilterOptions** - Interface defining the hook's configuration options
- **Search filtering** - Filters devices by name/hostname and OS type
- **Organization filtering** - Filters devices by selected organization IDs
- **organizationOptions** - Formatted organization data for dropdowns/selects

## Usage Example

```typescript
import { useDeviceFilter } from './use-device-filter'
import type { Device } from '../types/device.types'

function DeviceList({ devices }: { devices: Device[] }) {
  const {
    searchTerm,
    setSearchTerm,
    selectedOrgIds,
    setSelectedOrgIds,
    organizationOptions,
    filteredDevices,
  } = useDeviceFilter({ 
    devices,
    enabled: true // Set to false when modal is closed to skip org fetching
  })

  return (
    <div>
      <input
        value={searchTerm}
        onChange={(e) => setSearchTerm(e.target.value)}
        placeholder="Search devices..."
      />
      
      <select
        multiple
        value={selectedOrgIds}
        onChange={(e) => setSelectedOrgIds(Array.from(e.target.selectedOptions, opt => opt.value))}
      >
        {organizationOptions.map(org => (
          <option key={org.value} value={org.value}>
            {org.label}
          </option>
        ))}
      </select>

      {filteredDevices.map(device => (
        <div key={device.id}>{device.displayName || device.hostname}</div>
      ))}
    </div>
  )
}
```

The hook automatically fetches organizations when enabled and provides real-time filtering based on search terms (matching device names and OS types) and selected organization filters. It's optimized with memoization to prevent unnecessary re-renders during filtering operations.
