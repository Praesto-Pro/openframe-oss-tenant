A React hook that manages scheduled tasks for scripts, providing functionality to fetch and delete scheduled tasks filtered by script ID.

## Key Components

- **`ScheduledTask`** - Interface defining the complete structure of a scheduled task with timing configuration and actions
- **`ScheduledTaskAction`** - Interface for task actions that execute scripts with parameters
- **`scheduledTasksQueryKeys`** - Query key factory for consistent cache management
- **`useScheduledTasks`** - Main hook providing scheduled task operations for a specific script
- **API Functions** - `fetchScheduledTasksByScript` and `deleteScheduledTaskApi` for backend communication

## Usage Example

```typescript
import { useScheduledTasks } from '@/hooks/use-scheduled-tasks'

function ScriptScheduleManager({ scriptId }: { scriptId: string }) {
  const {
    scheduledTasks,
    isLoading,
    error,
    deleteTask,
    isDeleting,
    refetch
  } = useScheduledTasks(scriptId)

  const handleDeleteTask = (taskId: string) => {
    deleteTask(taskId, {
      onSuccess: () => console.log('Task deleted successfully'),
      onError: (error) => console.error('Failed to delete:', error)
    })
  }

  if (isLoading) return <div>Loading scheduled tasks...</div>
  if (error) return <div>Error: {error}</div>

  return (
    <div>
      <h3>Scheduled Tasks ({scheduledTasks.length})</h3>
      {scheduledTasks.map(task => (
        <div key={task.id}>
          <span>{task.name} - {task.task_type}</span>
          <button 
            onClick={() => handleDeleteTask(task.id.toString())}
            disabled={isDeleting}
          >
            Delete
          </button>
        </div>
      ))}
      <button onClick={() => refetch()}>Refresh</button>
    </div>
  )
}
```

The hook automatically filters all scheduled tasks to show only those that reference the specified script ID in their actions, and provides optimistic cache updates when tasks are deleted.
