A React component that provides an interface for scheduling script execution on selected devices with configurable parameters and timing options.

## Key Components

### Main Component
- **`ScheduleScriptView`** - Primary component handling script scheduling workflow with form validation, device selection, and API integration

### Form Schema
- **`scheduleFormSchema`** - Zod validation schema defining timeout, script arguments, environment variables, scheduling date, and repeat options
- **`ScheduleFormData`** - TypeScript type derived from the schema for form data structure

### Key Features
- **Device filtering and selection** - Multi-device targeting with organization-based filtering
- **Script configuration** - Timeout, arguments, and environment variable management
- **Scheduling options** - Date/time picker with optional daily repeat functionality
- **Batch execution** - Handles multiple device scheduling with success/failure reporting

## Usage Example

```typescript
import { ScheduleScriptView } from './schedule-script-view'

// Render script scheduling interface
function ScriptSchedulePage({ params }: { params: { scriptId: string } }) {
  return (
    <ScheduleScriptView scriptId={params.scriptId} />
  )
}

// Component handles:
// - Loading script details and available devices
// - Form validation with timeout (1-86400 seconds)
// - Device selection with filtering capabilities
// - Scheduled task creation via Tactical API
// - Error handling and user feedback via toasts
```

The component integrates with the Tactical RMM API to create scheduled tasks across selected agents, supporting both one-time and recurring executions with comprehensive error handling and user feedback.
