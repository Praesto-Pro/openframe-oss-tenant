A React component that displays execution history for scheduled scripts in a tabular format with detailed log viewing capabilities.

## Key Components

- **`ScheduleHistoryTab`** - Main component that renders a paginated table of script execution history entries
- **`ScheduleHistoryDrawer`** - Side panel component for viewing detailed log information including stdout/stderr output
- **`InfoField`** - Utility component for displaying labeled key-value pairs in the drawer
- **Helper Functions**:
  - `getStatusVariant()` - Maps execution status to UI color variants
  - `getStatusLabel()` - Formats status labels based on return codes and status
  - `formatDateTime()` - Formats ISO dates for display

## Usage Example

```typescript
import { ScheduleHistoryTab } from './schedule-history-tab'

function MySchedulePage() {
  const scheduleDetail = {
    id: 'schedule-123',
    name: 'Daily Backup',
    // ... other schedule properties
  }

  return (
    <div className="p-4">
      <ScheduleHistoryTab 
        schedule={scheduleDetail}
        scheduleId="schedule-123"
      />
    </div>
  )
}
```

The component fetches execution history using the `useScriptScheduleHistory` hook, displays entries in a clickable table with status indicators, and opens a detailed drawer when a log entry is selected. It includes pagination controls for large datasets and responsive design with column hiding on smaller screens.
