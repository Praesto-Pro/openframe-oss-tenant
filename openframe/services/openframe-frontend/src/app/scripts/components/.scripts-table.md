A React component that renders a filterable, searchable table of scripts with URL-persisted state management. It displays script information including name, shell type, supported platforms, category, and description with client-side pagination and filtering capabilities.

## Key Components

- **ScriptsTable** - Main component that manages state and renders the scripts table
- **UIScriptEntry** - Interface defining the structure of script entries displayed in the table
- **URL State Management** - Uses `useApiParams` to persist search, filters, and pagination in the URL
- **Data Transformation** - Converts raw script data to UI-friendly format with platform labels and tool types
- **Client-side Filtering** - Filters by search term, shell type, category, platform, and added-by fields
- **Responsive Design** - Columns hide/show based on screen size breakpoints

## Usage Example

```typescript
import { ScriptsTable } from './scripts-table'

// Basic usage in a page component
function ScriptsPage() {
  return (
    <div className="container">
      <ScriptsTable />
    </div>
  )
}
```

The component automatically handles:
- Fetching scripts data via `useScripts()` hook
- URL parameter synchronization for search, filters, and pagination
- Debounced search input (300ms delay)
- Row actions (edit, view details, run script)
- Mobile-friendly responsive layout with collapsible filters
- Navigation to script detail and edit pages
