A comprehensive table component for displaying and managing script schedules with search, pagination, and navigation functionality.

## Key Components

- **ScriptSchedulesTable** - Main component that renders a paginated table of script schedules
- **getRepeatLabelFromTaskType** - Utility function that converts task type enums to user-friendly repeat labels
- **Column Configuration** - Defines table columns for script name, date/time, repeat frequency, supported platforms, and device count
- **Search & Pagination** - Implements client-side filtering with debounced search and pagination controls
- **Navigation Actions** - Handles routing to schedule details, edit forms, and creation flow

## Usage Example

```typescript
import { ScriptSchedulesTable } from './script-schedules-table'

export default function SchedulesPage() {
  return (
    <div>
      <ScriptSchedulesTable />
    </div>
  )
}
```

The component integrates with the OpenFrame frontend core system, using:
- `ListPageLayout` for consistent page structure with search and actions
- `Table` component with responsive column hiding and row actions
- `OSTypeBadgeGroup` to display supported operating system platforms
- Custom hooks for API parameter management and table pagination
- Navigation to `/scripts/schedules/{id}` for details and `/scripts/schedules/create` for new schedules
