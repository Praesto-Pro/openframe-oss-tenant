A React component that displays schedule information in a grid-based card layout, providing both structured data view and raw data view variations for script schedule details.

## Key Components

- **ScheduleInfoBar** - Main component that displays schedule details (date, time, repeat pattern, supported platforms) from a `ScriptScheduleDetail` object
- **ScheduleInfoBarFromData** - Alternative component that accepts raw data props including schedule name and optional note fields
- **OSTypeBadgeGroup** - Imported component for displaying supported operating system platforms as badges

## Usage Example

```typescript
import { ScheduleInfoBar, ScheduleInfoBarFromData } from './schedule-info-bar'
import type { ScriptScheduleDetail } from '../types/script-schedule.types'

// Using with structured schedule data
function ScheduleView({ schedule }: { schedule: ScriptScheduleDetail }) {
  return (
    <div>
      <h2>Schedule Details</h2>
      <ScheduleInfoBar schedule={schedule} />
    </div>
  )
}

// Using with raw data
function CustomScheduleView() {
  return (
    <ScheduleInfoBarFromData
      name="Daily Backup"
      note="Critical system backup"
      date="2024-01-15"
      time="02:00 AM"
      repeat="Daily"
      platforms={['linux', 'windows']}
    />
  )
}
```

Both components feature responsive design with a 2-column layout on mobile expanding to 4 columns on desktop, consistent styling using ODS (OpenFrame Design System) classes, and proper text truncation handling.
