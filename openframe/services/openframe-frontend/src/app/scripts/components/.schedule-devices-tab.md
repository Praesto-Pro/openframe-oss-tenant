A React component that displays a table of devices assigned to a script schedule, showing device hostnames and OS types with responsive design.

## Key Components

- **ScheduleDevicesTab** - Main component that renders a table of scheduled agents/devices
- **useScriptScheduleAgents** - Custom hook for fetching schedule agent data
- **Table** - Reusable table component from OpenFrame UI library
- **OSTypeBadge** - Badge component for displaying operating system types
- **LoadError** - Error display component for failed data loading

## Usage Example

```typescript
import { ScheduleDevicesTab } from './schedule-devices-tab'

function SchedulePage() {
  const schedule = {
    id: 'sched_123',
    name: 'Weekly Maintenance',
    // ... other schedule properties
  }

  return (
    <div>
      <h2>Schedule Details</h2>
      <ScheduleDevicesTab 
        schedule={schedule} 
        scheduleId="sched_123" 
      />
    </div>
  )
}
```

The component features:
- **Responsive table** with device hostname and OS type columns
- **Error handling** with user-friendly error messages
- **Loading states** with skeleton rows during data fetching
- **Empty state** messaging when no devices are assigned
- **Result counter** showing total number of assigned devices
- **Mobile optimization** with column hiding on smaller screens (`hideAt: 'md'`)
