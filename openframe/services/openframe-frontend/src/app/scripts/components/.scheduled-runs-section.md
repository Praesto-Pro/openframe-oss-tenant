A React component that displays scheduled tasks for a specific script in a tabular format, providing an overview of when and how often scripts are scheduled to run.

## Key Components

- **ScheduledRunsSection** - Main component that renders the scheduled tasks table
- **formatDate()** - Utility function that converts ISO date strings to formatted date and time
- **getRepeatLabel()** - Utility function that converts task types to human-readable repeat intervals
- **ScheduledTaskRow** - Interface defining the table row structure
- **Table Integration** - Uses OpenFrame's Table component with custom column rendering

## Usage Example

```typescript
import { ScheduledRunsSection } from './scheduled-runs-section'

export default function ScriptDetailsPage({ params }: { params: { id: string } }) {
  return (
    <div>
      <h1>Script Details</h1>
      {/* Other script information */}
      
      <ScheduledRunsSection scriptId={params.id} />
    </div>
  )
}
```

The component automatically fetches scheduled tasks using the `useScheduledTasks` hook and displays them with formatted dates, repeat intervals, and device counts. It handles loading states, error states, and empty data scenarios through the integrated Table component.
