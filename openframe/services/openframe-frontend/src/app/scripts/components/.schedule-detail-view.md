A React component that renders the detail view for a script schedule, displaying schedule information, assigned devices, scripts, and execution history in a tabbed interface.

## Key Components

**Main Export:**
- `ScheduleDetailView` - Main detail view component that renders schedule information with tabbed navigation

**Configuration:**
- `SCHEDULE_TABS` - Tab configuration array defining three tabs: Scheduled Scripts, Assigned Devices, and Execution History
- Each tab includes an ID, label, icon component, and corresponding tab component

**Props Interface:**
- `ScheduleDetailViewProps` - Defines the required `scheduleId` string parameter

## Usage Example

```typescript
import { ScheduleDetailView } from './schedule-detail-view'

function SchedulePage({ params }: { params: { id: string } }) {
  return <ScheduleDetailView scheduleId={params.id} />
}

// The component handles:
// - Loading states with ScheduleDetailLoader
// - Error states with LoadError/NotFoundError
// - Navigation back to schedules list
// - Edit actions for devices and schedule
// - Tabbed content display with URL synchronization
```

The component uses the `useScriptSchedule` hook to fetch schedule data and provides action buttons for editing devices and schedule settings. It renders a responsive detail page with an info bar showing schedule status and a tabbed interface for different aspects of the schedule.
