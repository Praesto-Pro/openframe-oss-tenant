A React component that displays a tab showing scripts associated with a scheduled task, featuring expandable cards for each script action with detailed configuration information.

## Key Components

- **`ScheduleScriptsTab`** - Main container component that renders a list of script cards for a given schedule
- **`ScheduleScriptCard`** - Individual expandable card displaying script details including name, timeout, arguments, and environment variables
- **Props interfaces** - `ScheduleScriptsTabProps` and `ScheduleScriptCardProps` for type safety

## Usage Example

```typescript
import { ScheduleScriptsTab } from './schedule-scripts-tab'
import type { ScriptScheduleDetail } from '../types/script-schedule.types'

function ScheduleDetailsPage() {
  const schedule: ScriptScheduleDetail = {
    actions: [
      {
        script: 123,
        name: 'Database Backup',
        timeout: 300,
        script_args: ['--full', 'database=prod'],
        env_vars: ['BACKUP_DIR=/tmp/backups', 'LOG_LEVEL=info']
      }
    ]
  }

  return (
    <div className="p-6">
      <h1>Schedule Details</h1>
      <ScheduleScriptsTab schedule={schedule} />
    </div>
  )
}
```

The component features responsive design with mobile-optimized layouts, smooth expand/collapse animations using CSS Grid, and navigation integration to view detailed script information. Each script card displays timeout settings, parsed arguments, and environment variables in an organized format using InfoCard components.
