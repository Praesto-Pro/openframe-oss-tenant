A responsive form card component for configuring script actions within scheduled jobs, providing script selection, timeout configuration, and parameter management with platform filtering.

## Key Components

- **ScheduleActionFormCard**: Main form card component with responsive layout for mobile and desktop
- **ScriptPlatformIcons**: Helper component displaying platform compatibility icons for scripts
- **Props Interface**: `ScheduleActionFormCardProps` defining component requirements including script index, available scripts, platform support, and removal callback

## Key Features

- **Responsive Design**: Mobile-first layout with stacked elements, expanding to grid layout on tablets/desktop
- **Script Filtering**: Automatically filters scripts based on supported platforms and currently selected script
- **Form Integration**: Uses `react-hook-form` with Controller components for form state management
- **Dynamic Configuration**: Supports script arguments and environment variables when a script is selected
- **Auto-population**: Automatically sets script name and default timeout when a script is selected

## Usage Example

```typescript
import { ScheduleActionFormCard } from './schedule-action-form-card'

function ScheduleForm() {
  const scripts = [
    { id: 1, name: 'Backup Script', supported_platforms: ['linux', 'windows'], default_timeout: 120 },
    { id: 2, name: 'Monitor Script', supported_platforms: ['linux'], default_timeout: 60 }
  ]

  return (
    <FormProvider methods={methods}>
      <ScheduleActionFormCard
        index={0}
        scripts={scripts}
        supportedPlatforms={['linux', 'windows']}
        onRemove={() => removeAction(0)}
      />
    </FormProvider>
  )
}
```

The component handles script selection with platform compatibility indicators, timeout configuration, and dynamically shows script arguments and environment variable inputs when a script is selected.
