A skeleton loading component that provides placeholder UI elements while the Edit Script page loads, matching the exact layout and structure of the actual form.

## Key Components

- **EditScriptLoader** - Main loading component that renders skeleton placeholders for the entire edit script form
- **PlatformSkeleton** - Skeleton for platform selection cards and "Run as User" checkbox
- **FormFieldsSkeleton** - Skeleton for form fields (Name, Shell Type, Category, Timeout)
- **DescriptionSkeleton** - Skeleton for the description textarea
- **ArgumentsSkeleton** - Skeleton for script arguments and environment variables sections
- **EditorSkeleton** - Skeleton for the syntax/code editor area

## Usage Example

```typescript
import { EditScriptLoader } from './edit-script-loader'

function EditScriptPage({ params }: { params: { id: string } }) {
  const [isLoading, setIsLoading] = useState(true)
  
  if (isLoading) {
    return <EditScriptLoader />
  }
  
  return <ActualEditScriptForm scriptId={params.id} />
}
```

The component uses responsive grid layouts (2 cols on mobile, 4 on desktop) and maintains consistent spacing with the actual form. Each skeleton section corresponds to a specific part of the edit script interface, providing visual feedback during data loading.
