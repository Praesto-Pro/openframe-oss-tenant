A React component that provides a tabbed interface for viewing scripts and their schedules with URL-based tab state management.

## Key Components

- **ScriptsView**: Main component that renders a tabbed interface with two tabs: "Scripts List" and "Scripts Schedules"
- **SCRIPTS_TABS**: Configuration array defining the available tabs with their icons and associated components
- **Tab Management**: Handles URL synchronization for active tab state using Next.js routing

## Usage Example

```typescript
import { ScriptsView } from './scripts-view'

// In a page component
export default function ScriptsPage() {
  return (
    <div className="container mx-auto p-6">
      <h1 className="text-2xl font-bold mb-4">Scripts Management</h1>
      <ScriptsView />
    </div>
  )
}
```

The component automatically manages tab state through URL parameters (e.g., `?tab=list` or `?tab=schedules`) and renders the appropriate sub-component. The "Scripts List" tab shows the `ScriptsTable` component with a bracket-curly icon, while the "Scripts Schedules" tab displays the `ScriptSchedulesTable` component with a calendar icon. Tab changes update the URL without full page navigation using Next.js router replacement.
