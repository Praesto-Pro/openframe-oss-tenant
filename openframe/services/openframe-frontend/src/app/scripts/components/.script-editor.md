A React component that provides a Monaco Editor instance specifically configured for script editing with support for multiple shell types and a custom dark theme.

## Key Components

**Main Export:**
- `ScriptEditor` - Monaco Editor wrapper with shell-specific syntax highlighting

**Configuration Objects:**
- `SHELL_TO_LANGUAGE` - Maps shell types to Monaco language identifiers
- `ODS_THEME_NAME` - Custom dark theme identifier
- `defineOdsTheme()` - Configures the custom "ods-dark" theme with Flamingo brand colors

**Theme Features:**
- Dark background (`#161616`) with high contrast text
- Flamingo accent color (`#ffc008`) for keywords, cursor, and highlights
- Syntax highlighting for shell, PowerShell, Python, TypeScript, and batch files
- Custom scrollbar and selection styling

## Usage Example

```typescript
import { ScriptEditor } from './script-editor'

function MyScriptPage() {
  const [script, setScript] = useState('#!/bin/bash\necho "Hello World"')

  return (
    <ScriptEditor
      value={script}
      onChange={setScript}
      shell="bash"
      height="500px"
      readOnly={false}
    />
  )
}

// Read-only display
function ScriptViewer({ content }: { content: string }) {
  return (
    <ScriptEditor
      value={content}
      shell="powershell"
      readOnly={true}
      height="300px"
    />
  )
}
```

The component automatically maps shell types (bash, powershell, python, etc.) to appropriate Monaco language modes and applies consistent theming across all supported script types.
