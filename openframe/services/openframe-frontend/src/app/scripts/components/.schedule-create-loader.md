A loading skeleton component that mirrors the exact layout of the Schedule Create/Edit form while data is being fetched.

## Key Components

- **ScheduleCreateLoader** - Main export that renders the complete page skeleton
- **NameFieldSkeleton** - Loading state for the schedule name input field
- **DateRepeatSkeleton** - Loading state for date picker and repeat checkbox
- **PlatformsSkeleton** - Loading state for platform selection buttons (3 platforms)
- **ActionCardSkeleton** - Loading state for individual schedule action cards

## Usage Example

```typescript
import { ScheduleCreateLoader } from './schedule-create-loader'

function ScheduleCreatePage() {
  const { data, isLoading } = useScheduleQuery()
  
  if (isLoading) {
    return <ScheduleCreateLoader />
  }
  
  return <ScheduleCreateView data={data} />
}
```

The component uses the `Skeleton` component from the OpenFrame UI library to create animated placeholders that match the dimensions and layout of the actual form elements. Each skeleton section corresponds to a specific part of the schedule creation interface, ensuring a smooth loading experience that maintains visual consistency with the final rendered form.
