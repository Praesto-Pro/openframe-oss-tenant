A React component that displays detailed information about a specific script, including metadata, arguments, environment variables, and the script's source code in a read-only editor.

## Key Components

- **ScriptDetailsView**: Main component that renders a comprehensive script details page with navigation actions
- **useScriptDetails**: Custom hook for fetching script data by ID
- **ScriptInfoSection**: Displays script metadata (description, shell type, platforms, category)
- **ScriptArgumentsCard**: Shows script arguments and environment variables in organized cards
- **ScriptEditor**: Read-only code editor for displaying the script's source code
- **DetailPageContainer**: Layout wrapper with back button and action buttons

## Usage Example

```typescript
import { ScriptDetailsView } from './script-details-view'

// In a Next.js page component
export default function ScriptDetailPage({ params }: { params: { id: string } }) {
  return <ScriptDetailsView scriptId={params.id} />
}

// Or in a modal/drawer
function ScriptModal({ scriptId, onClose }: { scriptId: string; onClose: () => void }) {
  return (
    <Modal onClose={onClose}>
      <ScriptDetailsView scriptId={scriptId} />
    </Modal>
  )
}
```

The component handles loading states, error conditions, and provides action buttons for editing and running the script. It uses a responsive grid layout to organize script arguments and environment variables, and displays the script's source code in a syntax-highlighted editor.
