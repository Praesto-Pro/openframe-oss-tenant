A React component that displays comprehensive details for a single automation script, including metadata, execution options, and code syntax with synchronized scrolling.

## Key Components

- **`ScriptDetailsView`** - Main component that renders the complete script details page
- **Script Information Display** - Shows script metadata through `ScriptInfoSection`
- **Interactive Code Viewer** - Displays script syntax with line numbers and synchronized scrolling
- **Action Controls** - Provides run, edit, and schedule operations via buttons and dropdown menu
- **Conditional Sections** - Shows script arguments, environment variables, and scheduled runs based on data availability
- **Feature Flag Integration** - Conditionally displays scheduling features based on feature flags

## Usage Example

```typescript
import { ScriptDetailsView } from './script-details-view'

// Display script details page
function ScriptPage({ params }: { params: { id: string } }) {
  return (
    <ScriptDetailsView scriptId={params.id} />
  )
}

// The component handles all states internally:
// - Loading state with CardLoader
// - Error states with LoadError/NotFoundError  
// - Success state with full script details
// - Navigation to edit, run, and schedule pages
```

The component uses the `useScriptDetails` hook to fetch data and provides a rich interface for viewing and managing automation scripts, including synchronized scrolling between line numbers and code content for an optimal code reading experience.
