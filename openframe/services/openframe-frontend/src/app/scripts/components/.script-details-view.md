A React component that displays comprehensive details for a specific script, including its metadata, arguments, source code, and scheduled runs.

## Key Components

- **ScriptDetailsView**: Main component that renders the complete script details page with navigation, actions, and content sections
- **useScriptDetails**: Custom hook for fetching script data and handling loading states
- **DetailPageContainer**: Layout wrapper providing header, navigation, and action buttons
- **ScriptInfoSection**: Displays script metadata like description, shell type, and supported platforms
- **ScriptArgumentsCard**: Shows script arguments and environment variables in organized cards
- **ScriptEditor**: Read-only code editor displaying the script's source code with syntax highlighting
- **ScheduledRunsSection**: Conditionally rendered section showing scheduled script executions

## Usage Example

```typescript
import { ScriptDetailsView } from './script-details-view'

// Render script details page
function ScriptPage({ params }: { params: { scriptId: string } }) {
  return <ScriptDetailsView scriptId={params.scriptId} />
}

// The component handles all navigation actions internally:
// - Back to scripts list
// - Edit script
// - Run script immediately  
// - Schedule script (if feature flag enabled)
```

The component provides a complete script management interface with conditional features based on feature flags, responsive layout for arguments/environment variables, and comprehensive error handling for loading states.
