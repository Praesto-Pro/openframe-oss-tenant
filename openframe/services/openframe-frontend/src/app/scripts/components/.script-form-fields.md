A comprehensive form component for editing script configurations with platform selection, metadata fields, arguments, environment variables, and script content editing capabilities.

## Key Components

- **ScriptFormFields**: Main form component that renders all script editing fields using React Hook Form
- **Platform Selection**: Multi-select interface for choosing supported platforms with availability indicators
- **Metadata Fields**: Input fields for script name, shell type, category, timeout, and description
- **Dynamic Arguments**: Configurable script arguments and environment variables sections
- **Script Editor**: Integrated code editor for script content with syntax highlighting

## Usage Example

```typescript
import { useForm } from 'react-hook-form'
import { ScriptFormFields } from './script-form-fields'
import type { EditScriptFormData } from '../types/edit-script.types'

export function EditScriptForm() {
  const form = useForm<EditScriptFormData>({
    defaultValues: {
      name: '',
      description: '',
      shell: 'bash',
      category: 'General',
      supported_platforms: ['linux'],
      default_timeout: 90,
      run_as_user: false,
      args: [],
      env_vars: [],
      script_body: ''
    }
  })

  return (
    <form onSubmit={form.handleSubmit(onSubmit)}>
      <ScriptFormFields form={form} />
      <button type="submit">Save Script</button>
    </form>
  )
}
```

The component handles platform availability checking, responsive layout for mobile/desktop views, and integrates with the OpenFrame design system for consistent styling and user experience.
