A comprehensive form component for script editing that renders all necessary fields for script configuration and management within the OpenFrame platform.

## Key Components

- **ScriptFormFields**: Main form component that renders all script configuration fields
- **Platform Selection**: Multi-select interface for OS platform compatibility
- **Form Controls**: Name, shell type, category, and timeout configuration
- **Script Arguments**: Dynamic key-value pair management for arguments and environment variables
- **ScriptEditor**: Integrated code editor for script content with syntax highlighting

## Usage Example

```typescript
import { useForm } from 'react-hook-form'
import { ScriptFormFields } from './script-form-fields'
import { EditScriptFormData } from '../types/edit-script.types'

function EditScriptPage() {
  const form = useForm<EditScriptFormData>({
    defaultValues: {
      supported_platforms: [],
      run_as_user: false,
      name: '',
      shell: '',
      category: '',
      default_timeout: 90,
      description: '',
      args: [],
      env_vars: [],
      script_body: ''
    }
  })

  return (
    <form onSubmit={form.handleSubmit(onSubmit)}>
      <ScriptFormFields form={form} />
      <button type="submit">Save Script</button>
    </form>
  )
}
```

The component integrates with React Hook Form for validation and state management, providing a complete interface for script creation and editing with platform selection, shell configuration, argument management, and syntax-highlighted code editing.
