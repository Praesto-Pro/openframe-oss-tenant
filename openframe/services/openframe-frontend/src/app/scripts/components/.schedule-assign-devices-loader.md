A loading skeleton component that provides placeholder UI while the Schedule Assign Devices page loads, matching the exact layout and structure of the actual page.

## Key Components

- **`InfoBarSkeleton`** - Skeleton for schedule information bar with name/note (top row) and date/time/repeat/platform details (bottom row)
- **`SelectionModeSkeleton`** - Skeleton for selection mode radio buttons with descriptions
- **`DeviceRowSkeleton`** - Skeleton for individual device table rows with icon, device info, and action buttons
- **`ScheduleAssignDevicesLoader`** - Main skeleton component that combines all sub-skeletons to match the full page layout

## Usage Example

```typescript
import { ScheduleAssignDevicesLoader } from './schedule-assign-devices-loader'

// Show while schedule assign devices data is loading
function ScheduleAssignDevicesPage() {
  const { data, isLoading } = useScheduleAssignDevicesQuery()
  
  if (isLoading) {
    return <ScheduleAssignDevicesLoader />
  }
  
  return <ScheduleAssignDevicesView data={data} />
}
```

The skeleton uses responsive design with mobile-first breakpoints and matches the exact spacing, borders, and layout structure of the actual Schedule Assign Devices page for a seamless loading experience.
