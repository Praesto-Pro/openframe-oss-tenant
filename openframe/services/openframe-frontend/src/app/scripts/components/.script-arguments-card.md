A React component that renders script arguments in a formatted info card, parsing key-value pairs from argument strings and displaying them in a structured layout.

## Key Components

- **ScriptArgumentsCard**: Main component that accepts a title and array of argument strings
- **ScriptArgumentsCardProps**: TypeScript interface defining the component's props
- **InfoCard**: External component from `@flamingo-stack/openframe-frontend-core` used for structured data display
- **Label**: External component for styled text labels

## Usage Example

```typescript
import { ScriptArgumentsCard } from './script-arguments-card'

function DeploymentView() {
  const scriptArgs = [
    'environment=production',
    'region=us-east-1',
    'replicas=3',
    'debug-mode'
  ]

  return (
    <div>
      <ScriptArgumentsCard 
        title="Deployment Arguments"
        args={scriptArgs}
      />
    </div>
  )
}
```

The component automatically parses arguments with `=` delimiters into key-value pairs and handles arguments without values. It returns `null` if no arguments are provided, making it safe to use conditionally. The styling uses the Azeret Mono font with OpenFrame design system classes for consistent appearance.
