A React component that provides a unified interface for creating new scripts or editing existing ones with integrated testing capabilities.

## Key Components

- **EditScriptPage** - Main component handling both create and edit modes for PowerShell scripts
- **Form Management** - Uses `useEditScriptForm` hook for form state and validation
- **Test Integration** - Includes real-time test execution with device selection modal
- **Navigation** - Dynamic back button routing based on create/edit context
- **Error Handling** - Displays user-friendly error states for loading failures

## Usage Example

```typescript
// Create new script mode
<EditScriptPage scriptId={null} />

// Edit existing script mode
<EditScriptPage scriptId="script-123" />

// Component automatically handles:
// - Form initialization with existing data
// - Test run execution and output display
// - Save/update operations
// - Navigation between script list and details
```

The component integrates multiple custom hooks for data fetching (`useScriptDetails`), form management (`useEditScriptForm`), and test execution (`useTestRuns`). It provides a comprehensive script editing experience with live testing feedback and proper loading/error states.
