A React component that renders a paginated list view for managing security policies with search, filtering, and CRUD operations.

## Key Components

- **Policies Component** - Main list view component with search and table functionality
- **usePolicies Hook** - Custom hook for policy data management and API operations
- **Table Integration** - Configurable data table with columns, actions, and filtering
- **Navigation Actions** - Route handling for policy creation and detail views
- **Search & Filtering** - Debounced search with real-time filtering capabilities

## Usage Example

```typescript
// Basic usage in a page component
import { Policies } from './components/policies'

export default function PoliciesPage() {
  return <Policies />
}

// The component automatically handles:
// - Loading policies from the API
// - Search with 300ms debounce
// - Navigation to policy details
// - Creating new policies
// - Table filtering and sorting
```

## Features

- **Real-time Search**: Debounced search input with 300ms delay for optimal performance
- **Table Management**: Sortable columns, row actions, and filtering capabilities  
- **Navigation**: Integrated routing to policy details (`/policies-and-queries/policy/:id`) and creation (`/policies-and-queries/new-policy`)
- **Responsive Design**: Uses OpenFrame UI components with consistent styling
- **Error Handling**: Built-in error states and empty data messaging
- **Refresh Capability**: Manual data refresh functionality

The component leverages the `usePolicies` hook for state management and integrates with external table column definitions through `getPolicyTableColumns` and `getPolicyTableRowActions` utilities.
