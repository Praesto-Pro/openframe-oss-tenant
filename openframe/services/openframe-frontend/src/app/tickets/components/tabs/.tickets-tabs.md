A client-side React component that provides tabbed navigation for ticket management, allowing users to switch between current and archived chat views.

## Key Components

- **TICKETS_TABS**: Static configuration array defining the available tabs (current/archived chats)
- **getTicketsTab()**: Utility function to find a specific tab by ID
- **getTabComponent()**: Utility function to retrieve the React component for a given tab
- **TicketsTabNavigation**: Main navigation component with URL synchronization and tab switching logic

## Usage Example

```typescript
import { TicketsTabNavigation, getTabComponent } from './tickets-tabs'

// Basic usage with default behavior
function TicketsPage() {
  return (
    <div>
      <TicketsTabNavigation />
      {/* Tab content will be handled by the navigation component */}
    </div>
  )
}

// Custom tab change handling
function CustomTicketsPage() {
  const [activeTab, setActiveTab] = useState('current')
  
  const handleTabChange = (tabId: string) => {
    setActiveTab(tabId)
    // Custom logic here
  }

  const TabComponent = getTabComponent(activeTab)

  return (
    <div>
      <TicketsTabNavigation 
        activeTab={activeTab}
        onTabChange={handleTabChange}
      />
      {TabComponent && <TabComponent />}
    </div>
  )
}
```

The component automatically handles URL parameter updates and provides a clean navigation experience by clearing tab-specific parameters when switching between tabs.
