A React table component for displaying and managing chat dialogs with support for both current and archived states, featuring pagination, filtering, and bulk archive operations.

## Key Components

- **`ChatsTable`** - Main table component with cursor-based pagination and filtering
- **`CurrentChats`** - Wrapper for displaying active chat dialogs
- **`ArchivedChats`** - Wrapper for displaying archived chat dialogs
- **`ChatsTableProps`** - Interface defining component props including archive state and status filters

## Usage Example

```typescript
import { ChatsTable, CurrentChats, ArchivedChats } from './chats-table'

// Basic usage for current chats
function ChatsDashboard() {
  const [statusFilters, setStatusFilters] = useState<string[]>([])
  
  return (
    <CurrentChats 
      statusFilters={statusFilters}
      onStatusFilterChange={setStatusFilters}
    />
  )
}

// Direct usage with archive state
function ChatManagement() {
  return (
    <ChatsTable 
      isArchived={false}
      statusFilters={['OPEN', 'IN_PROGRESS']}
      onStatusFilterChange={(filters) => console.log(filters)}
    />
  )
}

// Archived chats view
function ArchivedChatsView() {
  return <ArchivedChats />
}
```

The component provides comprehensive chat management with organization lookup, real-time filtering, row-level navigation to individual chat dialogs, and bulk operations for archiving resolved conversations.
