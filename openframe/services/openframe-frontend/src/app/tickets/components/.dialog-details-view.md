A React component that displays detailed view for dialog conversations with real-time messaging, approval handling, and dual chat interfaces for client and admin interactions.

## Key Components

- **DialogDetailsView**: Main component managing dialog state and real-time updates
- **Chat Systems**: Dual chat interfaces for client (Fae AI) and admin (Mingo AI) conversations
- **Real-time Processing**: NATS subscription with chunk catchup for message synchronization
- **Approval Management**: Handles approval requests with approve/reject functionality
- **Status Controls**: Dialog status management (put on hold, resolve)
- **Message Processing**: Historical message processing with error handling

## Usage Example

```typescript
import { DialogDetailsView } from './dialog-details-view'

// In a Next.js page or component
export default function DialogPage({ params }: { params: { id: string } }) {
  return (
    <div className="h-screen">
      <DialogDetailsView dialogId={params.id} />
    </div>
  )
}

// Router usage
const router = useRouter()
router.push(`/dialogs/${dialogId}`)
```

The component integrates with multiple custom hooks for dialog management, real-time subscriptions, and message processing. It supports both client-facing (Fae) and admin (Mingo) chat interfaces with approval workflows and status management controls.
