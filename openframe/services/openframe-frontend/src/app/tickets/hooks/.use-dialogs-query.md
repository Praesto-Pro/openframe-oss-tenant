A React Query hook for fetching and managing dialogs with filtering, pagination, and caching capabilities. Supports archived/active filtering, status-based filtering, search functionality, and cursor-based pagination through GraphQL.

## Key Components

- **`useDialogsQuery`** - Main hook that accepts filtering parameters and returns dialog data with pagination
- **`DialogsQueryParams`** - Interface for query parameters (archived, search, statusFilters, cursor)
- **`DialogsResponse`** - GraphQL response structure containing dialog connection data
- **Status Logic** - Automatic status filtering based on archived state and custom status filters
- **Error Handling** - Comprehensive GraphQL and HTTP error handling with meaningful error messages

## Usage Example

```typescript
import { useDialogsQuery } from './hooks/use-dialogs-query'

function DialogsList() {
  const {
    dialogs,
    pageInfo,
    isLoading,
    isError,
    error,
    hasNextPage,
    endCursor,
    refetch
  } = useDialogsQuery({
    archived: false,
    search: 'customer inquiry',
    statusFilters: ['ACTIVE', 'ACTION_REQUIRED'],
    cursor: null
  })

  if (isLoading) return <div>Loading dialogs...</div>
  if (isError) return <div>Error: {error}</div>

  return (
    <div>
      {dialogs.map(dialog => (
        <div key={dialog.id}>{dialog.title}</div>
      ))}
      
      {hasNextPage && (
        <button 
          onClick={() => refetch({ 
            cursor: endCursor 
          })}
        >
          Load More
        </button>
      )}
    </div>
  )
}
```

The hook automatically handles caching (1-minute stale time), retries failed requests twice, and provides convenient pagination helpers for implementing infinite scroll or load-more functionality.
