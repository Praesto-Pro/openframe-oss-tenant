A React hook that fetches organization overview data with device statistics for dashboard display. It combines organization information with their device counts and status breakdowns.

## Key Components

- **`useOrganizationsOverview`** - Main hook that returns organization data with device statistics
- **`fetchOrganizationsOverview`** - Internal function that fetches organizations and aggregates device counts
- **`OrganizationOverviewRow`** - Interface defining the structure of each organization row with device metrics
- **`OrganizationNode`** - Type for raw organization data from the GraphQL API
- **`OrganizationsResponse`** - Type for the organizations GraphQL response structure

## Usage Example

```typescript
import { useOrganizationsOverview } from './use-organizations-overview'

function OrganizationsDashboard() {
  const { 
    rows, 
    loading, 
    error, 
    totalOrganizations,
    refresh 
  } = useOrganizationsOverview(20)

  if (loading) return <div>Loading organizations...</div>
  if (error) return <div>Error: {error}</div>

  return (
    <div>
      <h2>Organizations ({totalOrganizations})</h2>
      {rows.map(org => (
        <div key={org.id}>
          <h3>{org.name}</h3>
          <p>Total Devices: {org.total}</p>
          <p>Active: {org.active} ({org.activePct}%)</p>
          <p>Inactive: {org.inactive} ({org.inactivePct}%)</p>
        </div>
      ))}
      <button onClick={() => refresh()}>Refresh Data</button>
    </div>
  )
}
```

The hook automatically handles error states, caching (3-minute stale time), and provides comprehensive organization statistics including device status breakdowns and percentages.
