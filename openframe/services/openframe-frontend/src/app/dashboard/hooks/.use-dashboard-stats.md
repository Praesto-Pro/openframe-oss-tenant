React Query hooks for fetching and managing dashboard statistics data including device metrics, chat analytics, and combined dashboard state.

## Key Components

- **`useDevicesOverview()`** - Fetches device statistics with total, active, inactive counts and percentages
- **`useChatsOverview()`** - Retrieves chat analytics including resolution times and Fae AI rates (SaaS mode only)
- **`useSharedDashboardData()`** - Combines both device and chat data with unified loading states and refresh controls

## Usage Example

```typescript
import { useDevicesOverview, useChatsOverview, useSharedDashboardData } from './use-dashboard-stats'

// Individual hooks
function DevicesWidget() {
  const { total, active, activePercentage, isLoading } = useDevicesOverview()
  
  if (isLoading) return <div>Loading devices...</div>
  
  return (
    <div>
      <h3>Devices: {total}</h3>
      <p>Active: {active} ({activePercentage}%)</p>
    </div>
  )
}

// Combined dashboard data
function Dashboard() {
  const { devices, chats, isAnyLoading, refetchAll } = useSharedDashboardData()
  
  return (
    <div>
      <button onClick={refetchAll}>Refresh All</button>
      {devices.data.total} devices, {chats.data.total} chats
    </div>
  )
}
```

The hooks include automatic caching, error handling, and configurable retry logic. Chat statistics are conditionally enabled based on SaaS tenant mode.
