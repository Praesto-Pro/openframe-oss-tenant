A client-side React component that renders the main dashboard content with sections dynamically shown based on application mode (SaaS vs non-SaaS).

## Key Components

- **DashboardContent**: Default export component that orchestrates all dashboard sections
- **OnboardingSection**: User onboarding and getting started content
- **DevicesOverviewSection**: Device management and status overview
- **ChatsOverviewSection**: Chat/communication features (SaaS tenant mode only)
- **OrganizationsOverviewSection**: Organization management interface
- **isSaasTenantMode**: Utility function to determine if running in SaaS tenant mode

## Usage Example

```typescript
import { Suspense } from 'react'
import DashboardContent from './dashboard-content'
import DashboardSkeleton from './dashboard-skeleton'

// Dynamic import with loading state
export default function DashboardPage() {
  return (
    <Suspense fallback={<DashboardSkeleton />}>
      <DashboardContent />
    </Suspense>
  )
}
```

The component is designed for dynamic imports with loading skeletons, providing a smooth user experience while dashboard sections load. The chat section conditionally renders based on whether the application is running in SaaS tenant mode, allowing for different feature sets between deployment modes.
