Main dashboard content component that dynamically renders all dashboard sections based on the application mode, designed for client-side rendering with dynamic import support.

## Key Components

- **`DashboardContent`** - Default export component that orchestrates all dashboard sections
- **`isSaasTenantMode`** - Utility function to determine if chat features should be shown
- **Section Components**:
  - `OnboardingSection` - User onboarding interface
  - `DevicesOverviewSection` - Device management overview
  - `ChatsOverviewSection` - Chat functionality (SaaS mode only)
  - `OrganizationsOverviewSection` - Organization management overview

## Usage Example

```typescript
// Typically used with dynamic import for performance
import dynamic from 'next/dynamic'

const DashboardContent = dynamic(() => import('./dashboard-content'), {
  loading: () => <DashboardSkeleton />
})

export default function DashboardPage() {
  return (
    <div className="container mx-auto">
      <h1>Dashboard</h1>
      <DashboardContent />
    </div>
  )
}
```

The component conditionally renders the chat section based on SaaS tenant mode, making it adaptable for different deployment configurations. All sections are wrapped in a responsive layout with consistent spacing.
