A TypeScript barrel file that exports custom Vue composables for the Mingo AI system, providing centralized access to dialog management, chat functionality, and realtime subscription utilities.

## Key Components

- **`useMingoDialog`** - Single dialog management composable
- **`useMingoDialogs`** - Multiple dialogs management composable  
- **`useMingoDialogSelection`** - Dialog selection state management composable
- **`useMingoChat`** - Chat interface and messaging composable
- **`useMingoRealtimeSubscription`** - Real-time data subscription composable

## Usage Example

```typescript
// Import individual composables as needed
import { 
  useMingoDialog, 
  useMingoChat, 
  useMingoRealtimeSubscription 
} from './mingo-composables'

// In a Vue component
export default {
  setup() {
    // Manage a single dialog
    const { isOpen, openDialog, closeDialog } = useMingoDialog()
    
    // Handle chat functionality
    const { messages, sendMessage, isLoading } = useMingoChat()
    
    // Subscribe to real-time updates
    const { data, isConnected } = useMingoRealtimeSubscription('channel-name')
    
    return {
      isOpen,
      openDialog,
      messages,
      sendMessage,
      data
    }
  }
}
```

This barrel export pattern allows clean imports and better tree-shaking while organizing related Mingo AI composables in a single entry point.
