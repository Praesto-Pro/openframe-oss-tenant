A React hook for managing Mingo AI dialog interactions, providing functionality to create chat dialogs and send messages with error handling and state management.

## Key Components

- **useMingoDialog** - Main hook exposing dialog management functionality
- **createDialogMutation** - TanStack Query mutation for creating new ADMIN-type dialogs
- **sendMessageMutation** - TanStack Query mutation for sending messages to existing dialogs
- **State management** - Tracks current dialog ID using both state and ref for persistence
- **Error handling** - Comprehensive error states with toast notifications

## Usage Example

```typescript
import { useMingoDialog } from './use-mingo-dialog'

function ChatComponent() {
  const {
    currentDialogId,
    isCreatingDialog,
    isSendingMessage,
    createDialog,
    sendMessage,
    resetDialog,
    hasDialog
  } = useMingoDialog()

  const handleSendMessage = async (message: string) => {
    const success = await sendMessage(message)
    if (success) {
      console.log('Message sent successfully')
    }
  }

  const handleCreateNewChat = async () => {
    const dialogId = await createDialog()
    if (dialogId) {
      console.log('New dialog created:', dialogId)
    }
  }

  return (
    <div>
      {hasDialog && <p>Active dialog: {currentDialogId}</p>}
      <button onClick={handleCreateNewChat} disabled={isCreatingDialog}>
        Create Dialog
      </button>
      <button 
        onClick={() => handleSendMessage('Hello Mingo!')} 
        disabled={isSendingMessage}
      >
        Send Message
      </button>
      <button onClick={resetDialog}>Reset</button>
    </div>
  )
}
```

The hook automatically creates dialogs when sending messages if none exists, handles API errors with user-friendly toast notifications, and maintains dialog state across component re-renders using both useState and useRef for reliability.
