A React hook that manages infinite pagination of Mingo AI dialogs with real-time unread message tracking and search functionality. It combines GraphQL queries with local message state to provide a comprehensive dialog management interface.

## Key Components

- **`transformToDialogItem`** - Converts GraphQL dialog nodes into standardized `DialogItem` format with unread counts
- **`useMingoDialogs`** - Main hook that handles infinite query pagination, search filtering, and unread message integration
- **Query Configuration** - Configured with 30-second stale time and 60-second refetch interval for real-time updates
- **Message Store Integration** - Links with `useMingoMessagesStore` to track unread message counts per dialog

## Usage Example

```typescript
import { useMingoDialogs } from './use-mingo-dialogs'

function DialogsList() {
  const { 
    dialogs, 
    isLoading, 
    hasNextPage, 
    fetchNextPage,
    refetch 
  } = useMingoDialogs({
    search: 'support tickets',
    limit: 50,
    enabled: true
  })

  return (
    <div>
      {dialogs.map(dialog => (
        <div key={dialog.id}>
          <h3>{dialog.title}</h3>
          <span>Unread: {dialog.unreadMessagesCount}</span>
          <time>{dialog.timestamp.toLocaleString()}</time>
        </div>
      ))}
      {hasNextPage && (
        <button onClick={() => fetchNextPage()}>
          Load More
        </button>
      )}
    </div>
  )
}
```

The hook automatically transforms raw GraphQL responses into UI-friendly dialog items while maintaining pagination state and unread message synchronization across the application.
