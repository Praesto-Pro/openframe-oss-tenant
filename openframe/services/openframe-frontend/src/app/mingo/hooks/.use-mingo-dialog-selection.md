A React hook that manages Mingo dialog selection, message loading, and approval workflows with infinite scrolling and real-time status updates.

## Key Components

- **Dialog Management**: `selectDialog` mutation for switching between dialogs with loading states
- **Message Loading**: Infinite query system that fetches admin messages with pagination
- **Approval System**: `handleApprove` and `handleReject` functions with optimistic updates
- **Status Tracking**: Real-time approval status management across dialog switches
- **Data Processing**: Converts GraphQL messages to historical format using core utilities

## Usage Example

```typescript
import { useMingoDialogSelection } from './use-mingo-dialog-selection'

function MingoDialogInterface() {
  const {
    selectDialog,
    isLoadingMessages,
    handleApprove,
    handleReject,
    approvalStatuses,
    rawMessagesCount
  } = useMingoDialogSelection()

  const handleDialogSelect = (dialogId: string) => {
    selectDialog(dialogId)
  }

  const handleRequestApproval = async (requestId: string) => {
    await handleApprove(requestId)
    // Status automatically updates in store and UI
  }

  return (
    <div>
      {isLoadingMessages ? (
        <div>Loading messages...</div>
      ) : (
        <div>
          <p>Messages loaded: {rawMessagesCount}</p>
          {/* Render messages with approval controls */}
        </div>
      )}
    </div>
  )
}
```

The hook integrates with Zustand store for state management and provides error handling with toast notifications for approval operations.
