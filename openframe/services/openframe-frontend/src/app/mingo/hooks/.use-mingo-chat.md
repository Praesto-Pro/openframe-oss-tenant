A React hook that provides comprehensive chat functionality for Mingo AI interactions, including message management, dialog creation, and approval handling within the Flamingo MSP platform.

## Key Components

- **useMingoChat** - Main hook that manages chat state and provides chat operations
- **ProcessedMessage** - Interface defining the structure of processed chat messages
- **UseMingoChat** - Return type interface defining all available chat functionality
- **Message filtering** - Logic to separate regular messages from pending approvals
- **Optimistic updates** - Immediate UI updates before server confirmation

## Usage Example

```typescript

  }

  const handleCreateDialog = async () => {
    const newDialogId = await createDialog()
    if (newDialogId) {
      // Navigate to new dialog
    }
  }

  return (
    <div>
      {messages.map(msg => (
        <div key={msg.id}>{msg.content}</div>
      ))}
      {isTyping && <div>Mingo is typing...</div>}
      {approvals.length > 0 && (
        <div>Pending approvals: {approvals.length}</div>
      )}
    </div>
  )
}
```

The hook integrates with Zustand state management, TanStack Query for server state, and provides automatic message filtering to separate chat content from approval requests. It includes optimistic updates for better UX and comprehensive error handling with toast notifications.
