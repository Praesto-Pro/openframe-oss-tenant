A Next.js page component that implements the Mingo AI chat interface with real-time messaging, dialog management, and sidebar navigation for technical support conversations.

## Key Components

- **Main Component**: `Mingo()` - The primary chat interface page component
- **State Management**: Manages active dialog, draft chat mode, and URL synchronization
- **Hooks Integration**: Utilizes multiple custom hooks for dialog management, messaging, and real-time subscriptions
- **Real-time Subscriptions**: `DialogSubscription` components for NATS-based message processing
- **Layout Structure**: Three-column layout with sidebar, message list, and input area

## Usage Example

```typescript
// This is a Next.js page component that's automatically routed
// URL: /mingo or /mingo?dialogId=123

// Key user interactions:
// 1. Start new chat
const handleNewChat = () => {
  // Creates draft chat mode
  setIsDraftChat(true)
  setActiveDialogId(null)
}

// 2. Select existing dialog
const handleDialogSelect = (dialogId: string) => {
  // Updates URL and activates dialog
  router.replace(`/mingo?dialogId=${dialogId}`)
  setActiveDialogId(dialogId)
  subscribeToDialog(dialogId)
}

// 3. Send message
const handleSendMessage = async (message: string) => {
  if (isDraftChat) {
    // Creates new dialog first
    const dialogId = await createDialog()
    await sendMessage(message, dialogId)
  } else {
    await sendMessage(message)
  }
}
```

The component enforces SaaS tenant mode, handles keyboard shortcuts (Escape to close), and provides real-time message synchronization across multiple dialog subscriptions.
