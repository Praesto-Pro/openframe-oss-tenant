API service layer for Mingo chat operations that provides React Query mutations for dialog management, message handling, and approval workflows through the centralized apiClient.

## Key Components

- **MingoApiService** - Static class providing mutation factories for Mingo chat operations
- **createDialogMutation()** - Creates new admin chat dialogs
- **sendMessageMutation()** - Sends messages to existing dialogs  
- **approveRequestMutation()** - Approves approval requests with toast notifications
- **rejectRequestMutation()** - Rejects approval requests with error handling
- **TypeScript Interfaces** - Type definitions for API requests and responses

## Usage Example

```typescript
import { MingoApiService } from './mingo-api-service'

function ChatComponent() {
  const createDialog = MingoApiService.createDialogMutation()
  const sendMessage = MingoApiService.sendMessageMutation()
  const approveRequest = MingoApiService.approveRequestMutation()

  const handleCreateChat = async () => {
    try {
      const dialog = await createDialog.mutateAsync()
      console.log('Dialog created:', dialog.id)
      
      await sendMessage.mutateAsync({
        dialogId: dialog.id,
        content: 'Hello Mingo!'
      })
    } catch (error) {
      console.error('Chat operation failed:', error)
    }
  }

  const handleApproval = async (requestId: string) => {
    await approveRequest.mutateAsync(requestId)
    // Toast notification handled automatically
  }

  return (
    <div>
      <button onClick={handleCreateChat}>Start Chat</button>
      <button onClick={() => handleApproval('req-123')}>Approve</button>
    </div>
  )
}
```
