A utility class for logging JUnit 5 test execution summary statistics using SLF4J logging framework.

## Key Components

- **`logSummary(TestExecutionSummary summary)`** - Static method that logs test execution statistics including tests found, succeeded, and failed counts

## Usage Example

```java
import org.junit.platform.launcher.Launcher;
import org.junit.platform.launcher.LauncherDiscoveryRequest;
import org.junit.platform.launcher.core.LauncherFactory;
import org.junit.platform.launcher.listeners.SummaryGeneratingListener;

// Create launcher and discovery request
Launcher launcher = LauncherFactory.create();
LauncherDiscoveryRequest request = LauncherDiscoveryRequestBuilder
    .request()
    .selectors(selectPackage("com.openframe.test"))
    .build();

// Execute tests with summary listener
SummaryGeneratingListener listener = new SummaryGeneratingListener();
launcher.registerTestExecutionListeners(listener);
launcher.execute(request);

// Log the summary
SummaryLogger.logSummary(listener.getSummary());
```

The class provides a clean way to output test execution results to logs, making it useful for CI/CD pipelines and automated testing workflows where you need visibility into test outcomes.
